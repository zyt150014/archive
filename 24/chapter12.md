# chapter12 去中心化

# 一个主要的中心演变成多个小的中心，这也就是去中心化的简单含义。
# 在谈论去中心化时，我们需要先了解中心化。这也很好理解，中心化也是在社会发展中将某些公共概念趋于统一的产物。在“书同文，车同轨”的中央集权下，减少了个体之间的交流障碍，促进了物质的流通和经济的发展。于此同时，普通的个人也往往希望能有一套完整的秩序，大家都能通过遵从这个秩序减少各种生产生活成本，这也是积极的一方面。
# 除了共识的中心化之外，数据也天然地趋于中心化。不论是在不同地区搜集数据的传感器，还是个人使用电子设备接入互联网，又或者只是生活区域的一份调查报告，最终都将数据存放的载体在物质空间中聚集起来，以更好地去分析数据以及数据的更快速流通。中心的存在依赖于流入数据的数据源的信任，也是发展过程中的必经道路。
# 而事物总是存在正反两面。中心化的趋势往往会对自身进行强化，以巩固自身的中心地位，这也被称之为垄断。而劣币驱逐良币则是在劣币凭借中心化的趋势，以先发优势相对于良币占据了优势地位，后续又利用优势地位扩大先发优势，最终完全占据大部分市场。
# 而这种优势地位也有可能转变成劣势。对于一个已经占据了大部分市场的中心事物来说，这种劣势的转变来自于每个个体对其的信心缺失。这种信心有多个方面的因素影响，而个体感觉受到了欺骗则是最有可能打击信心的情况。随着大部分个体的信心下降，也会加速中心地位的倒塌过程。但一个中心的倒塌往往是另一个中心形成的开始，寻找另一个替代品是一种普遍选择。这种倒塌-重建的过程也是一种客观规律，正如屠龙者终将成为恶龙的循环。
# 让我们再来看看去中心化的含义。既然中心化并不完美，存在着各种各样的问题，那么我们可以选择降低规模，选择去中心化吗？
# 区块链是我在去中心化时联想到的第一个概念。当然你可以去研究一下区块链的相关原理，如果我提及到一些更加具体的区块链的组成元素，那也表明我在先前了解了相应的概念。但是我并不了解区块链，所以也不会去提及里面的东西。去中心化的本质是依赖客观的可信共识在个体之间的交流活动。
# 首先，去中心化本身不受个人感情的影响。物质空间的中心化运作依赖于某些特定的个人和群体，而人参与的因素也会对这种中心化的运作产生影响。其次，去中心化可以不被追踪，个人之间的交流不需要引入第三个人。
# 但是在物质空间中对去中心化的使用使得这里并没有完全实现去中心化，由区块链诞生的虚拟货币作为一种应用其最可靠的交易仍然是通过第三方交易所。而这里也不存在完全客观理性，也不会被打击的交易所，个人之间的交易也需要经受信任的考验。现在的去中心化更多的是逃离现有中心化的思想来驱动的。
# 为什么我们会选择中心化？因为这是社会发展的客观结果。中心化可以提高个人对中心化的信任，提高发展效率，从而使大多数人从中受益，也是大多数个体的选择。而去中心化的发起则是出于少部分人对于中心化的不信任。在中心化集中程度越来越高之时，它对每个个体的影响也越来越大，这种影响力的提升会提高个人利用中心化谋取利益的概率。在全体利益产出确定的情况下，个人谋取的额外利益必然来自被这种中心化影响力所影响的个体，这也值得我们关注。
# 而当下的去中心化也存在这个问题。在不能实现完全的中心化之下，当前的去中心化以小规模的中心化存在。这种小规模的中心化可能出于对某个概念的共识，也可能是某些个人谋取利益的工具。在将大的中心化分化成小的中心化之时，谁会在这个过程中得到利益？
# 现实世界在中心化和去中心化之间动态调整，以符合大多数人的认知和发展利益。在趋于中心化时我们可能会偏向去中心化的自由，在趋于去中心化时我们也可能会偏向中心化的统一。而在这个过程中，我们也可能到达一个相对平衡的状态。
# 去中心化对个人提出了要求，你需要对真与假进行更准确的认知判断。
