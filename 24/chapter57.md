# chapter57 层次

# 在组合的过程中，也就形成了自下而上的层次。
# 物质空间中的实践活动，在层次的上层与下层进行着不同的实际工作。在整体的功能出现问题时，自上而下或者自下而上进行的检查工作让我们可以分析出问题所在。找到了功能不正常的联系之后，就可以在这一个层次进行更加深入的排查工作了。这样的流程在层次概念中是典型的沟通方式，也具有普适性。
# 继续了解一下物质层面的层次概念。自下而上，按照空间中的相对大小我们可以进行层次划分。将个人所在的层次作为主要观察对象，这一层次之上与之下都有着相应的联系。个人所在的层次在个人的认知与实践过程中扩展了各种概念，并将这些概念延伸出更多的概念。个人认知下的层次之间也并没有绝对的界限，上下之间的交流工具也在不同层次之间起着沟通作用。更加严格与比较宽泛的标准会产生不同的层次种类，但我们也只会在我们所在的这一层产生意义。
# 对于比较接近的下一层次，我们有着更多的操作空间。我们可以在自由意志下使用各种各样的工具进行特定的实践活动，同时这些工具也会在固定的操作下产生预期的结果，这也就是流程的复制再现过程。而对于同一层次的概念，则既有着更多的操作空间也有着更多的条件限制。具有同样自由意志的个人和其他的生物在合作与竞争中形成了特定的默契，无形的边界所形成的范围在整体上的妥协成为了整体的解决方案。随着层次的上升，相对大小也逐渐变大，我们也逐渐丢失了控制能力。如果有着更高层次的意志，那么我们也没有任何可能性去改变这种意志的方向。在尚未发现之时，客观层面的物质演化也足以让我们产生敬畏。
# 转而思考思维层面的层次含义。查看物质空间层次含义的投影时我们也会发现同样的联系，但在这里我们则有着完全的自主性。只要思维层面所形成的概念之间不产生相悖的逻辑过程，那就可以进行完全设想。在大部分情况下可能并没有什么用，但更多的概率性在意外收获中也不是一件坏事。
# 思维层面同样可以进行层次的自下而上或者是自下而上的分析流程，已经被应用过的物质层面的投影之外也可以进行同样的处理。将这些元素进行语义化，然后进行自下而上的语法分析。以终结符作为解释终点，在定义的语法中我们进行各种替换尝试。同一种具体概念使用一套语法如果存在多个解释结果，那么这种具有二义性的语法也不太能作为标准来进行使用。在此基础上增加特定的解释过程可以消除这种歧义，从而成为解释其他语义的准则。特定语义在经过这套解释准则处理之后所形成的语法树也天然地具有自身的层次，这也是这种语义不同层次的直接体现。
# 可分析的语义经过层次解析后形成了静态的符号以及这些符号之间的静态联系。将各种各样的静态处理器置于动态的输入时，结合形成的整体也就成为了更大的层次概念。再思考一下自上而下的分析过程。面对已经完善的整体我们需要进行层次的拆解工作，这也就是组合的逆运算。对于同一个整体，不同的拆解方法也有着各自的优劣，个人如何选择则是一个值得思考的问题。将下一个层次再次进行拆分之时，具体概念的层次深度也需要与其他参与组合的元素进行整体上的评估，个人认知刷新之时也有了划分层次的更好方案。
# 将层次分析应用于各种层次的整体时，我们会形成不同类别的层次理解。在理论上，这里的层次种类在有限之时也必然会投射到具有不同外在表现的概念中。对应的实际过程则是个人在具体事物层次方面进行思考时获得了先前已经有过的熟悉感觉，这种意识也会催生层次组合种类的总结过程。相似的组合模式作为不同整体的部分在自身产生的效应上也存在相似之处，个人的明确分辨会产生更加全面的认知。
# 如果我们先射箭再画靶，那就可以说我们永远都生存在相关的现实意义层次之上。实际上也是如此，如何扩展这个边界就是剩下的问题了。
