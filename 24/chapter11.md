# chapter11 数据

# 在近乎无限的数据中，个人总存在感知范围。
# 在先前已经提及的物质空间和虚拟世界，都是数据的一种呈现方式。存在于物质空间中的某个具体概念，在每时每刻都有一种特定的状态，我们可以用一系列的数据将这个物质概念的这个状态在客观上表示出来。不妨让我们把从抽象的最初始的状态发展到这个状态所经历的状态改变数量作为第一参数，当下我们无从得知具体的数值，但是它就在那里。组成这个物质概念的物理子的各种状态，从微观层面到宏观层面的各个阶段的不同呈现，以及和所有都在此状态的其他概念之间的相对关系，从第二参数开始向下不断延伸。这里的参数位置也是一种数据信息，在解释的时候用来被告知这一位参数的含义。
# 虚拟世界则是由数据来驱动的抽象概念。在美好的虚拟世界中，感受器的所有感受都是作为外设的计算机的计算结果。感受器也将这些计算数据经过一层映射关系转化成电信号数据，最终在大脑中形成一种感觉数据。我们的感受器在当前还能够在输入的数据中分辨出虚拟世界和物质空间的区别。而不论是虚拟世界的真实性的增强，还是个人感受器的能力削弱，都有可能产生物质空间和虚拟世界的混淆现象。
# 人的大脑在客观上也会相对性地忽视一些数据，从而更加注重另一些数据。这在一方面是人的大脑运行效率低，记忆内容容易丢失，数据的输入输出也比较慢，另一方面也是因为在个人的层面上，不同的数据有着不同的重要性。我也只会记得昨天晚上我在什么地方吃了什么，对于地点的环境和食物的外观口感，我也只有一个模糊的印象，我也不会去过度在意各种没有被记录载体所写入的细节。
# 而在数据的记录和分发工具得到了很大改善的现在，我们产生的无用的数据也越来越多，这也需要使用重要性的过滤器来进行处理。既然这个信息是一种公共概念空间的数据，那就有机会被用来做一些其他的事情。最常见的现象就是，在数据的分发阶段，为了提高数据的影响力，使用在公共概念空间中一般而言的更重要的概念对本来是一般重要的概念进行替换，而在替换之后让我们来对比看一看相关的结果。如果将一个包含重要性的概念的数据信息和另一个有着不太重要的概念的数据信息放在你面前，你会选择哪一个？群体的概率性已经表明了答案。个人通过相对比较重要的概念去进行深入了解这个数据信息的概率性要高于不太重要的概念，这仅仅是因为重要性这一个参数。利用这个我们可以将一些更加不重要的概念和表面上重要的概念进行打包，从而达到提高某些不重要概念的影响力。
# 这种利用方式也称得上是“上有政策，下有对策”，但是只要是在相对合理的范围内也无可指摘。这里也顺便提及一下另一种方式，也就是对数据的预测的利用。上面进行的重要性的替换在现实意义上比较难去将其控制在相对合理的范围内，也就是“常在河边走，哪能不湿鞋”，而如果我们直接去掉这个重要性又会发生什么？
# 我们也会天然地去对不完整的含义去进行补充，这是一种语义预测。在公共的概念空间中，这些不完整的含义的补充也有一份概率列表。重要性的替换可能只是这个概念在概率列表中的小幅度提升，而个人的补充则很有可能直接填入概率列表中的最高概率选项。当然这种方式不一定有效，还需要了解概率列表中不同概率的概念的重要性，和重要性的替换相比利用起来也更加复杂。
# 而另一种利用个人惯性的推荐算法更加流行。在针对多个有相似的使用数据的个人进行大规模的分析之后，这种数据转而用来强化个人惯性。这些都是对数据的使用方式。
# 状态的改变不仅产生了时间概念，同时也在进行数据的积累。数据分析产生的价值，很大程度上来源于这些数据背后暗藏的个人行为模式的识别。而在你看着各种屏幕时，数据的排列组合在你眼中便具有了意义。在物质空间中一直都存在的半导体材料中产生的电子数据，构成了虚拟世界中产生的各种概念。
