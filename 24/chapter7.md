# chapter7 相对性

# 在从整体的角度来认知和实践之后，使用相对性分析可以用来研究不同的个体。
# 相对性的含义也很好解释：在对多个对象进行评估判断时，我们选择其中某个特定的对象，然后以此为基础，与其他的对象进行对比。相对性存在的原因是同一时间存在多个具有共性的事物，相对性也就是这些事物之间的一种关联方式。
# 而当我们在个人概念空间中查看相对这个抽象概念的关联时，我们也会有不同的概率列表。在这里我最先想到的是相对论，这也是爱因斯坦在现代科学中巨大影响力的体现。当然相对论在专业性和文字蕴含的信息量上从相对的角度比其他的很多参考资料都要高，在通过了至今众多的观测验证之后也表现一致，我们因而可以以此继续构建物理大厦。但我个人并不探究物理，对相对论的了解也是浅尝辄止，毕竟没有人生来就注定要从事某个事业，而对一个概念追根溯源的过程是无穷无尽的。在明白了需要了解的基本知识之后，我们以自身需要来动态地进行选择。
# 相对性作为进行认知所得个人概念空间中的概念之间的一种关联，这种关联也因人而异。举个例子：现在桌子上有一杯水，水还剩一半，这作为前提。一个乐观的人看到之后说：“幸好还有半杯水”，而另一个悲观的人看到之后说：“糟糕只剩一半了”，这是从感性方面的认识。一个非常口渴的人看到之后一饮而尽，仍觉得不够，而另一个人却正好喝完，这是从个人需要出发的判断。一个人看到后也可能会认为应该换个小一点的杯子，能正好装满水，另一个人看到后会觉得桌子太空了，应该多放几杯水，这是从桌子，杯子，水三者之间进行的相对考虑。
# 从相对性出发，得到的结果往往可以从二元角度来进行解释。我们可以继续看看上文的水的例子，感性方面的认识也就是个人对世界看法的积极性倾向程度的多少，个人需要出发的判断就是和特定数值的大小比较，对桌子，杯子，水三者之间进行的相对考虑是从整体到局部的尺度占比。以这些不同的参考来进行度量，可以形成二维中的参数线。参数线包括闭合和开放两种，尺度占比就属于闭合的参数线。这些不同的结果属于不同的线上的点，足够多的点之间形成的关联在整体上又可以被视为一种概念。
# 随着事物不断变化，相对性的标准也在随之变化，这也说明完全基本上都会变得相对。我们可以使用各种各样的相对概念，组合成一件合格产品。在制造一批机器时，首先需要组成机器的所有零件。在所有零件中，只要符合相对误差的需求，那就算是合格的零件。但是这里会有一些问题：两个零件都在各自的误差范围之内，但是在把这两个零件组合起来使用时这两个的误差偏向是二元对立的方向，这就导致这两个零件的组合体并不合格。
# 现在我们可以讨论一下这里的几种解决办法。其一：我们可以承认相对性的合格标准并不是一成不变的，过去的标准也可能不适合当下。在放宽整体的需求之后，也就是我们作为裁判调整了裁定规则，那么这两个零件也就是合格的。其二：如果我们需要去坚持相对性的标准，那可以选择对这两个零件进行打磨，使这些零件更加精确，组合起来也能满足合格性的需求。其三：在这一批机器中，对于同一个零件有不同的相对误差的实例。我们可以将其中一个零件与其他机器的零件进行调整调换，在排列组合的选择中找到所有机器都合格的选项。以上三种方案都有各自的优缺点，需要根据实际来进行权衡。其一放宽标准可能导致对标准的破坏，其二打磨零件可能会影响零件的其他方面，其三可能无法找到一个满足所有条件的组合方式。但最简单的方式也很明显：在跳出固定的数量限制之后，我们可以允许零件的相对冗余，从而相对地提高这批机器的合格率。
# 在机器生产出产品之后，如何去划分每个零件的贡献？在这个过程中，不存在一个完全确定的1+1=2的算法。即使我们可以搜集到足够多的数据，最多也只能保证相对合理。而在你注意到这一点之后，你也在相对性地忽视其他未曾注意到的东西。
# 相对而言，相对性需要一定的关注度。
