# chapter49 娱乐

# 在个人的形形色色的情感变化中，娱乐产生的愉悦让人着迷。
# 在提及到娱乐的具体含义时，对于不同的个人也会有不同的表现形式。在通常情况下，我们可以说出各种各样的娱乐活动，而在这些具体的活动实验中大部分人都能在情感上产生积极的变化过程，所以这被划分到娱乐活动的范畴。在这里的实验中的少部分人，对于某些特定的娱乐活动可能并不感兴趣。不同的人对于娱乐的不同偏好，在数据的显现中成为了个人画像。
# 如果我们已经搜集到了足够的数据，那么我们也就可以用这些数据用来做一些事情。将不同的数据结果进行对比，这种不同之处也显示出了不容易发现的细微差别。个人在经历了这种分析过程之后也就复制了个人的部分数据，从这些零散的数据中可以重新推导出模糊的个人印象。
# 在提供娱乐的一方看来，这种数据的搜集让自身对于不同的个人喜好有了更多的了解，在同一个竞争环境中会产生更多的优势。在消费娱乐的一方看来，个人可能在无意识中让第三方获得了个人不想进行分享的数据，在比较轻度的程度上与个人的意愿相反。提供者与消费者在很多情况下并不能达成一致，但提供者在自身对于所提供的娱乐的更加充足的了解让自身占据了客观的优势。
# 与个人的工作相辅相成，娱乐也具有自身的积极意义。在大脑进行了一段时间的活动之后，生理上的疲劳会带来理解与反应速度的降低。在适度的放松与娱乐的时间中个人对自身的精神状态进行一定的调整，从而更好地进行后续的思考过程。劳逸结合作为老生常谈也有可取之处。
# 在其他的一些情况下个人也可能会去选择娱乐。在个人需求中，一般的规律是个人对于尚未达到的需求选项拥有达到的动力。如果我们在使用了大多数动力的状态下可以在大概率中实现这些需求，那么这就成为了大多数人的期望而不是空中楼阁。就像将巨石从山脚推到山顶一样，未到山顶就重新滚下山来的单次结果在逐渐地消耗个人的希望而增加了个人的失望。在无休止的重复过程中，作为实验样本的群体在心理上崩溃的概率也在不断增加。这种积累过程也让个人需要时间来进行缓解，不去思考这些事情的转换思维过程中的娱乐也正好符合个人的需要。
# 但娱乐与工作也具有一些共同之处。在我们明确地将某段时间用于工作或者用于娱乐之时，这两种不同的活动也都需要个人在状态改变的过程中进行主动参与。如果这里所进行的娱乐活动在个人的层面上并不会有任何积极方面的效果，那么这可能也不足以被称为娱乐活动。这种实际感受也最先被消费娱乐的个人所感知，而这二者之间的距离也并非泾渭分明。在不断重复的娱乐过程中所带来的逐渐缩减的愉悦感受，最终也会让自身失去其娱乐属性。
# 所以在广义上我们如何理解娱乐过程？
# 基于个人先前的所有经历搭建起来的处理系统，在面对不同的事情时会产生不同的感受，同样也代表了不同的愉悦程度。旧有的经验在新事物之上的分析结论在新的设计中产生的效果也具有独特性，而新的体验也是娱乐在这样的情形之下所产生的效用。喜新厌旧的特性作为组成个人的基本要素，新事物及其衍生的整体在概念上也是一种娱乐。
# 在影像逐渐代替文字作为主流文化对个人的思想认知产生影响之时，波兹曼在《娱乐至死》中表达了自身的悲观想法：个人在心甘情愿中成为了娱乐的附庸，我们也终将毁于自己所热爱的东西。从得出的这种观点我们可以进行一些推导，来拥有一个更加全面的认知。在状态的改变过程中所形成的个人娱乐程度的曲线，在向上的愉悦与向下的平静中会逐渐提升阈值。为了再次拥有的美好感受个人的行为也会逐渐变形，在毁灭的漩涡中无法自拔。利用新的事物所带来的娱乐感受，在一定程度上可以缓解这个过程，但这也并不是解药。与越来越密集的概念空间类似，可以产生娱乐感受的新的事物的形成在客观层面上也会在娱乐的提供者才思枯竭的过程中逐渐消失，这也会让个人走上同一兴奋的程度加大过程。
