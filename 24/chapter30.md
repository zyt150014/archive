# chapter30 物质交换

# 在物质世界中，状态改变的一个重要过程就是物质交换。
# 对于我们来说，有意义的物质交换发生的空间位置处于地球表层上下的一定范围内，所以我们也只需要去研究这个范围之内的物质交换。在这个范围内，首先可以考虑从外界引入的太阳的能量。而太阳无时无刻不在进行聚变反应，同时也在巨大的压力之下保持着内外的平衡。这种聚变反应产生的能量的一小部分到达了地球表面，成为了地表物质交换活动的动力来源。
# 最开始利用太阳发出的光的光子所携带的能量的是在生物圈中的生产者。最原始也数量最多的生产者是各种藻类，这些藻类结构简单，利用光子能量的途径简短，其中的物质交换也不复杂。这种单细胞或者是由有限的细胞组成的多细胞生物在没有完整的叶绿体结构之时直接使用细胞质中的叶绿素完成能量的固定，而能量的固定则是我们熟知的二氧化碳和水转变成有机物和氧气的过程。这种能量的固定源于化学反应中的化学能变化，在低化学能的物质转变成高化学能的物质的过程中吸收了光所携带的能量，而这种高化学能的物质则在后续的状态改变中被用来控制物质的空间位置与维持物质的当前状态。
# 而在生产者的进化过程中，这种能量的固定效率也在提升。首先单个完整的生产者个体的不同部分之间的分工也趋向明确，加工原料的收集，物质结构的维持，不同部分之间的物质运输，以及最终的生产加工过程都在有条不紊地运行。在白天阳光充足之时，这种生产者也会利用水的蒸发来带动自底向上的水分输送。同样在夜晚时，也会调整自身的状态为第二天的生产加工过程进行准备。高效率也是自然所选择的方向。
# 在合成了越来越多的有机物之后，将有机物和氧气转变成水和二氧化碳等无机物的过程则是形成物质交换的循环的必要过程。各种消费者在这个物质交换的过程中重新获得了先前生产者所保存的太阳光的能量，同时也利用这些能量进行控制物质空间位置和维持物质状态的活动。
# 在消费者的物质交换过程中，人作为消费者也是这种过程的参与者，我们可以对自身的消费过程进行一些讨论。处于某个状态的我们，当前组成个人的所有物质可以进行数量上和类别上的分析，来了解不同元素的不同占比，同时也可以进行空间上的相对位置的分析，来形成各种空间上的不同结构。而再加上不同物质交换的前后状态标记时，这里所说的某种状态就有了唯一性。在下一个时间点，个人在呼吸过程中从外界获取氧气，并输出二氧化碳，将生产者所生产的有机物转变成无机物的过程中维持心脏的跳动和大脑的思考过程，个人身体的各部分肌肉也在这个过程中消耗能量用来维持组成个人的物质在空间上的相对位置的相对稳定的状态。这种呼吸过程是完全的消费过程，所以个人也需要从外界获取各种有机物来维持整体上物质总量和类别的稳定。进食和排出过程则是个人与外界进行物质交换的另一个重要过程，我们在整体的层层过滤中获取自身所需并在最后连同各种代谢产物一起输送到外界。
# 这种过程我在先前称为个人这种组合体维护自身有序的过程，这也是维持个人的熵在一个稳定的范围内的主要手段。而如果个人的熵不断增大，那么其也失去了控制物质的能力，最终会在生理层面上而死亡。
# 尘归尘，土归土。组成个人的物质在形成了一定的结构，也就是物质之间的相对位置保持稳定的状态后，这种组合体也就有了将这种稳定的状态保持下去的趋势。如果我们将组成个人的所有物质种类统计出来，并搜集外界的同种物质并将其限制在一定范围内，那么个人和这个范围内的物质在物质层面没有区别。而在变蝇人这部恐怖电影中，作为科学家的男主对于物质的研究也达到了非常成熟的地步，在影片中制造出来了将组成个人的所有物质进行分解并在另一个空间位置重新组合起来的机器。这种机器最简单的使用方式就是实现了个人的瞬移，如果为真则有更多的用法等待挖掘。而在一次实验中，一只苍蝇的意外进入导致了男主和苍蝇在重新组合的过程中被组合到一起，因而变成了半昆虫半人的怪物。这种物质之间组合方式的改变也是后续物质交换过程的一种研究方向。
