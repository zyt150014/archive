# chapter18 注意力

# 注意力作为个人关注更多的方面，你所需要的就是注意力。
# 在讨论注意力之前，我们可以先来看看个人的生理特点。在我们的眼睛聚焦到某个事物上时，其周围的事物会增加一层模糊状态，在你切换到另一个事物时也是一样。在某个存在状态时，在你的思考中会有名为当下的概念。你的注意力关注于当下，已经关注过了的事物向后延伸，前方则是即将产生关注的事物。而个人的大脑也不擅长同时处理多个不同的任务，这里正在处理的任务也依赖于注意力所关注的事物。
# 大脑的运行效率不够高，容易疲劳也是个人的生理特点，在头晕眼花的时候我们的注意力也会涣散。生活中各种事物的发生也都有可能吸引个人的注意力，在分散的注意力下我们几乎不能进行任何实践活动，集中注意力进行流水线工作是普遍的选择。
# 在某些情况，注意力会有意想不到的效果。数学上我们会去使用“注意到”作为后续数学推论产生的前提，这也是进行推断的关键条件。这种注意到也可能是对于一个前提已经有了足够的认知，在关注另一个问题时可以直接使用而不是重新回顾与现在的问题关系不大的长篇大论。注意到和未注意到也是我们注意力相对集中程度的大小，未注意到在之后的重要性也可能值得我们去注意到。
# 从注意力来看，个人的认知方式是单点聚集加上多点采样。单点的聚集让我们能更高效地投入实践活动，多点采样可以在个人层面上产生不同的中断信号，来由个人进行不同处理，这也是我们在面对不同的突发情况的应对方式。无论是窗外的汽车开动的声音，还是在你眼前游荡的飞蚊，在客观层面上你的感受器不会丢失它们在你大脑中产生的感受，在个人的过滤器中你可以选择转而去关注这些事物，也可以选择无视它们的发生继续集中注意力。
# 这是搭建具有独立思考能力的个体的过程中比较重要的一个步骤，不同的注意力参数组成了这个个体输入输出通道上的特定过滤器。在此之上是更为复杂的决策分析过程，在对注意力有了足够的了解之后我们就可以继续向上研究。
# 从主观的注意力出发，我们可以转向客观的注意力。在不讨论个人的进行不同处理的过滤器之后，在多个事物上的注意力差别与这个事物和其他事物之间的关联以及这些关联与公共概念空间中群体对特定的概念所保持的注意力有关。这种注意力的差别既是一种客观的体现，也受研究群体的特征影响。不同事物之间的关联，形成的是一种与研究群体特征无关的层次结构，这种结构中不同概念的相对位置是相对固定的，这也就是客观。研究群体的特征又确定了不同概念之间的相对注意力的高低，这也就是影响的结果。
# 虽然客观的注意力也会被群体的特征影响，但对此研究的第一步则是纯粹的事物之间的关联。这种客观的关联我们也知道它就在那里，在充满误差的认知下我们对它的了解与它本身的差别只会减小不会消失。而在概念空间中，我们可以在给定的条件下对取自现实的概念之间建立注意力关联，这也是一种语义中的注意力区别。
# 接下来可以看看在语义分析中的注意力。首先，这种注意力来自于对整体语义的思考。如果我在前文和后文都提及到了某个概念，那么无论是在前文进行分析还是在后文中研究关联这种注意力都会在对二者都已经存在了解的前提下来计算注意力。整体语义的思考包含着全部细节，你可以从更多的维度看到同一个事物的投影结果。其次，注意力和注意力之间也存在一种模糊的模式关联。在建立起一种概念的整体语义上的注意力的全息影像时，对另一种类似的概念进行思考可以从中得到一些启发，这也是语义空间中不同关联的相对固定的相对位置的表现。
# 在语义分析之后，我们就有了一份注意力的思考结果。随之而来的全部排列组合的可能性，在这个结果之下会在被评分之后在某个评分基准上下进行浮动。
# 注意力也是你在这个过程中使用的概念，你应该也注意到了。
