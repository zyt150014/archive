# chapter25 个人选择

# 不同的个人有不同的选择偏向。
# 在做出这个判断时，我们基本上也会同意这句话。不同的人有不同的经历，这些经历也就是影响个人选择偏向的因素。如果我们能有着同样的经历，我们也会做出一样的选择吗？
# 这个问题思考起来是比较有趣的。在我们刚来到人世的时候，我们可以认为我们都是一张白纸，我们的个人的概念空间都从零开始进行构建。但客观上我们的物质状态是不同的，有的头发多一点，有的脸更圆一点，这也就是作为对应的白纸的形状和大小之间存在差别。而在物质世界中也并不存在一样的环境，即使是双胞胎在空间上的相对位置也是不同的。每个人在不同的成长环境中会受到不同的影响，在这个过程中也逐渐形成了个人的概念空间，然后在不同的选择问题上形成不同的选择偏向。但是在现在的这个物质世界中，我们无法做出相关的实验来进行验证。
# 在这种情况下，我们可以借助一些事物来进行研究。对于一台固定配置的计算机，在一定的时间范围内我们认为它的经历是一样的，也就是在进行了一些计算给出了对应的结果时不会对后续的计算产生影响。在这段时间范围的相对早和晚的时间点，如果我们给出同样的计算问题，计算机会给出同样的回答，当然这里不会去使用随机数。而事实上无论在什么时间点，对于一台完整的并且正常运转的计算机，再给出不包含随机数的问题时都会给出一样的回答。这似乎表明在平行世界拥有同样经历的两个人，在一个问题上会做出一样的选择。
# 然而个人的大脑的构造要更加复杂，简单的计算机结构可能不具有解释性，我们可以再看看对语言模型的研究。在使用了众多的参数对模型进行训练之后，对于某个给定的训练好的模型，在我们给出相同的输入之时模型也会给出相似但不完全一样的答案。但是这种情况可能也并不能去解释这个问题，因为这种语言模型是一种概率模型。
# 这个问题在瑞克和莫蒂里面也有体现。在其中的一段情节中，瑞克和莫蒂因为暂停了莫蒂的爸爸和妈妈的时间，所以在恢复时间流动之时可能会变成各种可能性的碎片。而在这种不稳定的时空中，因为一些原因同一个时空产生了分裂，形成了两个同样的时空。在时空刚分裂之时，瑞克就确定了时空产生了分裂，而不同时空的瑞克也做出了一样的举动。而在瑞克准备使用时间水晶将分裂的时空重新拼接成一个时空时，这两个时空的瑞克和莫蒂产生了不同的选择，这也导致了恢复时空的失败。
# 当然在后续的剧情中瑞克和莫蒂在编剧的自由意志下让破碎的时空重新完整，在这里我们不再进行讨论。而上述我们所提到的情节，我觉得比较接近选择问题的答案。在一步步的选择中，如果将有着相同经历的个人分成多份，这多个人会因为个人的概率性去选择不同的选项。对于差别比较大的不同选项，选择同一个选项的概率就比较大，也就容易保持一致。而在差别很小的不同选项之上，就很容易产生不同的选择。
# 这种个人选择的概率性在很多情况下不会让个人的人生轨迹产生比较大的偏移，但在某些关键的时间点的选择则足以产生影响一生的变化。而我们当时可能并没有意识到这一点，而在经历巨大的变化之后才明白。
# 在这种情况下也没有完全正确的选择，在当时面对选择的个人对这个选择产生的长远影响也难以预测。而无论个人如何选择，这都是个人所认为的正确选择。在你的选择过程完成之后，你也只应该去做接下来的选择。
# 在个人对相似的问题进行选择时，去参考他人的选择是一种常见的选择。在包含所有的个人的概念空间的公共概念空间已经十分稠密的情况下，在物质世界中有众多的相似案例可供参考。在你检视不同的选择案例时，你也会逐渐确定自己的真正选择。但如果个人完全采用他人的选择，那么就失去了作为个人独特性的自由的思考。但这也是一种个人的选择，每个人只需对自己的选择负责，也不用去影响他人所做出的选择。
# 个人选择的序列来自于思考层面的自由。
