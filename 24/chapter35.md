# chapter35 文学

# 与客观的科学不同，文学具有主观上的特点。
# 以语言或者文字作为媒介的文学，是个人用来表达自身认知与进行交流的手段。语言或者文字的这种媒介，如果是用来进行状态的变化的一种记录之时，那么它就是物质世界的客观体现。
# 首先我们可以谈论一下记录式的文学。这种文学起源于史官对于已经发生之事的记录，在后续的发展中也包括航海日志和新闻稿。这些记录往往采取不掺杂个人情感的记录方式，在摆出尽可能全面的事实之时将评判的权利交给读者。
# 这种方式显然有它的两面性。在正面上，这些文字的解释权并不在这些文字的书写者手里，对这些文字的不同理解都是可以接受的。在反面上，失去了书写者作为接触这些文字的第一人视角的看法，也让这些文字的部分价值产生了缺失。
# 在个人情感的影响下，使用客观的记录加上个人对此的评论欲望，或者进行纯粹的虚空构造，就是体现了各种情感的文学作品。
# 这些文学作品也以各种形式存在，这也被称为文学体裁。不同的存在形式有着不同的格式要求，这种格式也是划分新生文学作品的一种标准。以诗歌为例，从诗经开始，每首诗都存在固定的韵律和格式。在四言的基本句式之外，也存在三言，五言，七言等格式，如知我者谓我心忧，不知我者谓我何求以及投我以木桃，报之以琼瑶等。在后续中五言诗和七言诗在唐朝得到了长足发展，形成了绝句与律诗等具有不同要求的文学形式，既包括平平仄仄平平仄，仄仄平平仄仄平，也有首联颌联颈联尾联等概念。随着后续思考形式的多样化，文学形式也更加广泛。个人既可以遵守某种特定的文学形式进行仿古创作，也可以随心所欲以各种形式来实现自身的表达。
# 而在谈论文学之时，通俗和高雅作为对立的两种概念，我们也可以思考一下这二者的演变轨迹。在个人将大部分时间用于劳作时，通俗的文学形式发展缓慢。在部分人有了额外的时间进行认知与实践活动后，人为设计的格式给这些文学增加了高雅的元素，而不同的常用语也是这种差别的体现。受众的局限性让这种文学的影响力被限制在一个范围内，所以将这些文学通俗化也是自然的趋势。白居易作为著名的诗人，在传闻中他在写完诗后，会将其读给老妪并询问老妪是否能理解。如果老妪尚不能理解，则继续修改直到老妪也能从诗里面了解到白居易所表达的情感。通俗的文学增加了受众，在更多的具有自由和独立思考能力的主体的思考中文学的概率性也得到了提升。
# 文学的书写着也往往也会受到其他文学的书写者的影响。在文字的所有排列组合方式中，如果个人使用了先前的人所使用过的排序方式，那么读者从中感受到的情感也会从先前的人的文学作品迁移过来，而这也就是抄袭的一种。而对于借鉴和抄袭，这里也并不存在明确的界限。
# 我们可以继续思考一下这种文学雷同。如果是原封不动的搬运，那么即使是客观的文字统计都能发现。而将某个文学作品作为参考对象，在理解和解构的基础上，再结合自身的独特属性，这也大概率不会被认定为抄袭。但历史的发展是一直向前的，后人也无法选择是否站在前人的肩膀上，因为前人的肩膀已经就在那里。这既是后人的一种幸运，也是一种不幸。一方面后人在某个具体的文学方向上有了充足的参考资料，另一方面在进行新文学的创作时也需要回避这些主题，在遍布机关的房间中找到通往出口的路线。
# 这也让我不由得对自己现在所进行的实践活动进行反思。文章本天成，妙手偶得之。在无穷无尽的状态改变过程中，所有客观的概念的排列组合方式都已经固定了下来，从个人思考而来的具有创造力的文学作品也只是在寻找某个前人尚未发现的唯一的排列方式而已。如果将当下的语境进行冻结，那么从第一章到这一章的所有文字在管理者的数据统计中总会在某个时间被人所发现，而我也只是找到这种唯一的排列方式的一人。但语境并不会冻结，概念也不是一成不变，在某个时间点产生的新的概念会极大增加排列方式的可能性，后续的所有章节也是在已经确定的排序中继续增加个人色彩的方法，我也在这个过程中获得了完全的乐趣。
# 文学形式映照出的个人，不同的情感也随之传递。
