# chapter36 艺术

# 建立在美之上的一种表现形式，即为艺术。
# 根据上文的定义，我们需要首先知道什么是美。然而美作为一种主观的感受，我们有不同的评判标准，个人的局限性同样也限制了我们的的判断。但对于具体的事物，它的美也可以成为一种普遍的共识。意大利画家达芬奇所创作的油画《蒙娜丽莎》，反映了文艺复兴时代特征的同时，也反映了人们在美的同一个方向上的追求。在一般情况下，公共概念空间中对于美的要求为在有意义的前提下的有序与协调。在有序与协调中，感受器官能得到放松，也就可以产生美的感觉。
# 这种有意义的前提是感受主体存在的条件。在通常情况下，我们也会认为火山喷发和海啸在某种程度上是一种美的体验，但如果个人置身其中则会让这种美的体验产生长时间的中断。有序与协调则是组成这种表现形式的不同部分之间的互相作用，在自然中产生的美而形成的艺术。
# 在当下的语境中，艺术根据表现形式的不同可以按照不同的模式来进行划分。这既包括上篇所讲述的文学，也包括音乐，舞蹈等使用不同知觉所获得的感知。因为视觉在个人从外界获取信息的途径中占比最大，所以在不同的艺术中使用视觉途径的也显然最多。结合不同的途径形成的艺术能让人获得更加全面的感受，这也是一种演变趋势。电影在结合了视觉和听觉这两大感知途径之后，在上个世纪蓬勃发展，在不同的时代也在讲述不同的故事。更进一步，在结合了视觉与听觉这两大途径的基础之上，增加个人的实践过程选项，让个人可以进行自由的选择，这也就是这个世纪继续发展的艺术形式————电子游戏。很明显的是，沿着这条路径，我们也能知道后续的艺术发展方向，也就是结合更多的感知途径，创造出更加接近现实的虚拟世界。
# 在毕达哥拉斯时代，艺术的美来自于组成整体的元素与整体的不同层次之间的大小比例。以数学形式所表述的个人头身比，使用不同比例长度的弦所组成的琴，这种和谐的视觉与听觉在美的感受下也是艺术的起源。
# 我们可以再分析一下这种表现形式。在这种整体上，我们总是可以将其分为外在和内在两个方面进行讨论。艺术的外在，也就是个人感受器官的直接感知，是在个人的概念空间中产生艺术内在认知的全部过程。外在作为艺术的皮，是艺术的重要组成部分；内在作为艺术的骨，是艺术得以存在的前提条件。在艺术的创作中，为了表达同一个艺术内核，我们可以采取不同的艺术形式，这种方式也十分常见。无论是将文学翻拍成电影，还是采用电子游戏的世界观进行二次的绘画和文学创作，都增加了个人感知同一个表达主题的维度，这也让个人有了更加丰富全面的体验。
# 如果表达的艺术空有一个美丽的外壳，却缺乏一个明确的指向，那么这种艺术则大概率来源于个人的功利性。这里也并不是完全不需要功利性，因为艺术的创作者在付出了个人的劳动而产生的艺术在能带给个人美的感受之时是有价值的。这种价值不容易去量化，这种美的感受也难以使用另外的艺术形式来进行替代。
# 在艺术反映时代特征之时，中间其实存在着审美概念。审美概念是一种相对性的概念，在我们进行二择来做选择题之时也就是审美概念发挥效果的时机。这种审美概念也遵循前文所描述的个人概念与公共概念的联系逻辑，随着状态的改变而不断演变。不同方面的审美也会带来不同的结果：三寸金莲在违反了个人的自然发育之时在宋代之后被称为美，当代则在健康的认知下以自然为美。但这里的自然就是最好的吗？
# 我们也可以思考一下，有序和协调为什么会产生美的感觉？
# 有序意味着存在额外的过程，而这种过程也正在将无序转变为有序。协调意味着组成整体的元素与整体之间的完整联系，这种联系让这个整体产生了生机。这二者所展现的，也正是生命力的美。同样具有生命力的个人，也正在将这种美保存在艺术之中。
# 艺术就是漫天繁星，我们也正在闪耀。
