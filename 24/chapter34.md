# chapter34 习得

# 在客观的层面上，没有什么是不能习得的。
# 这里的习得也是一种统称，是个人在所有感受器的刺激中所获得的全部信息，这种感觉或者是知觉产生的所有信息可以用不同的来源来进行划分。而一种不太可信的占比数据显示，在人脑从外界获取的所有信息的不同知觉的占比分别为：视觉83%，听觉11%，嗅觉3.5%，触觉1.5%，味觉1%。这种十分准确的数字在每个个体身上通常不太准确，但这些知觉之间的相对占比则足以作为参考。视觉作为个人从外界获取信息的最重要的途径，绝大多数信息都经由眼睛这个器官进入个人的大脑。听觉则是第二重要的感受器官，综合视觉与听觉个人就能对物质空间的具体事物产生绝大部分的认知。
# 从这些所有信息中，我们也首先会去过滤大部分在个人角度没有太大意义的信息，这也是先前所提到的对于相对而言更加重要的信息我们会给予更多的关注。在这些数据经过过滤之后才正式作为个人学习的材料，而在这些材料之中能被个人所理解与进行相关的推导的认知则又是其中的一小部分。在整个过程中，我们也会发现个人的所有可习得的东西在所有可被认知的概念之中的占比微乎其微。
# 习得的概念，我们也可以进行更加细致的分解，两种主要的区别就是在潜移默化中的习得与在特定的系统化的学习方案之下的习得。前者发生于个人清醒与睡眠的大多数时间，在各种感受器默认开启的前提下，个人在海量的信息之中会进行一种类似于哈希算法的过程，在这个过程中将大量的信息压缩到个人的独特属性中从而形成独特的个人色彩。这种压缩过程在个人的早期对个人的认知具有重要的影响，作为交流工具的语言的形成也是在这种影响之下搭建起最初的基础结构。随着个人概念空间在早期的构建，后续这种过程对于个人的影响力也逐渐减弱，这种被称为个人的本性的概念也往往被认为难以改变。而对于后者这种特定的系统化学习，我们也很熟悉。从小开始的各种教育，就是在特定方向的一种整体性培训。这种培训具有重复与固定的字眼的特点，而客观的要求也偏向于死记硬背。这种重复的低效训练在更多的情况下也只会阻碍个人对更大的物质世界的认知，也只是在这种内卷中获得少量的优越感。
# 在个人对物质世界的概念进行习得之后，也有两种不同的结果。
# 其一，我们在拥有了习得所组建而成的概念空间与思考体系之后，在后续的实践活动中对所习得的概念的验证符合个人的理解，从而加强这种习得所构建的个人概念空间。其二，如果这种实践与预期不符，而不论是物质世界中的不符还是个人理解错误所导致的不符，都是习得性无助产生的来源。
# 这种情形在个人的客观行为中时常发生。在你写了一篇文章，或者是对某些事物的结构做了一下改造，如果结果是正向的，也就是其一的情形，这会让这种习得获得物质世界的验证，也会增加可信度真实的概率。而如果结果偏离预期，这种反向也会作用于个人。在多次的同样的状态改变过程中，习得性无助是一种无可奈何的无望。这个方面的研究来源于上世纪六十年代美国心理学家塞里格曼所做的实验。首先他把一条狗关在笼子里并施加不可逃避的电击，在多次实验之后打开笼子的门，在继续的电击实验之后狗已经丧失了解决办法的希望，直接倒在地上等待痛苦的来临。塞里格斯在实践中得出了科学的结论：对于具有独立思考能力的主体在持续的失败情形中所学习到的无助的感觉，这种不具有解决办法，无可救药的判断也会被迁移到新的情形。
# 当然，我们需要去避免这种消极的习得，同时也需要对这种习得有着清醒的认识来避免陷入这种情形。在个人有着强烈的信仰与坚定的决心之时，客观的习得性无助也就无法动摇个人的精神支柱。在老人与海中，名叫圣地亚哥的老渔夫在长时间钓鱼而无所获之时仍充满着奋斗的精神。在钓到大马林鱼之后，两天两夜的全力战斗最终以渔夫的胜利写下了句号。尽管在返航途中大鱼未能逃脱被鲨鱼吃光的命运，但留下的鱼骨头也是故事开始到结束的所有状态的记录。
# 不同的习得也造成了不同的阶级。知识与概念的共享让我们都有了习得的权利，而如何把握就在个人的手中。
# 习得的所有概念，组成了个人的样貌。
