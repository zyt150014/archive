# chapter41 金融游戏

# 在货币的交易行为中，确定了足够正确的交易方向，剩下的也就是计算交易的时机与多少。
# 这也可以被称为是量化交易。使用不同的交易策略，不同的个人在这个过程中会得到不同的结果。在一定的有效范围内进行讨论时，对于货币以及特定的商品之间的兑换比例可以进行升降分析。在给定价格分布之后，我们总能计算出最优的交易策略。不过这种策略需要完全的信息，在概率性中我们也无法预测所有的事件，但是整体的趋势在对现状有了充足的了解之后也就呼之欲出。使用这种趋势得到的比较好的交易策略，也就已经超过了大多数人。
# 而这种操作作为纯粹的交易游戏，在实际影响上在虚拟经济中可能属于比较缺乏价值的一类。与可以进行低成本复制而产生情感价值的艺术相比，首先这就不具有拷贝的价值，因为在物质世界中不存在完全一样的操作对象；其次这种参考效益也大多数来源于后续的同一区域，这种行为在整体上带来的效益可能只有个人在这种参与中获得的情绪体验价值。
# 然而这真的就是这样吗？我们完全可以进行设想。这种货币的转移虽然在整体上不参与货币的增长过程，但这也是先前所说的认知价值的体现。在个人的能力与其可以造成的影响力相匹配之时，这种组合价值最终也会推动经济的发展。
# 我们可以再看一下经典的复利游戏在数学中的作用效果。伯努利在研究银行利率之时假设年利率不变的情况下，在个人增加付息次数之时所得利润也会随之增加，并且付息次数越多获得的相对利润也越多。而在计算这个数学表达式的值之时伯努利也发现是收敛的，这也是自然对数e的定义。在大多数人都了解了这种操作方式之后，人为的限制则被用来缝补体系的漏洞。在不同的时间对利率进行调整，对于不同的存取时间长度采用不同的利率，都是在先前市场反映的数据的前提中所优化的结果。
# 个人投资作为金融游戏的重要一环，客观的资产具有更好的认同价值。股票市场作为经济发展到一定程度形成的用来进行重新分配的产物，同样建立在社会的共识之下，并遵守法定规则。公司进行股票发行时，市场上的个人就可以购买相应的股票从而成为公司的投资者，并依靠股票获取对应的股息以及红利。经营良好的公司发行股份的目的是进行融资，并使用获得的资金进行更大的发展，并将发展所获得的成果与先前的投资者进行共享。这种状况来自于认知与资金的错位，掌握了发展机会的主体与拥有资金的主体之间没有重合，这也是非常常见的情况。如果行为主体自身就足以满足自身的需求，那可能就失去了发行股票的动力，这也同时失去了外界对于主体经营方向的影响。当局者迷，旁观者清，塞翁失马又焉知非福。
# 而在这种金融游戏下，个人的思考也会带来投机行为的发生。在个人使用抵押物的情况下，借入资产并且约定一个具体的到期时间，在这个到期时间到来之时个人可以使用各种方式来使用资产所代表的本金获取收益。如果个人并没有获得收益，那显然就会付出相应的代价。而在个人获得了超过被借入资产的主体的收益期望之时，这之间的差值或者是其他约定的特殊分配规则也就是双方获取的总收益。这里被借入资产的主体在约定的相应法律效果下借出资产，并在一定时间之后获取收益，这与从银行获取利息是同样的内在逻辑。而很显然的一点是，同样时间之内这种行为的收益率也会更高，但同时也并不是完全可靠。更高的回报率也往往意味着更多的风险，这种概率性衍生而来的赌徒心理也在人性之中找到了一条捷径。
# 如果扩大投机行为的规模，则可能会造成没有人能承受的后果。杠杆本意是指阿基米德撬动地球的手段，在金融游戏中成为了以小博大的工具。在缺乏相应监管手段之时，利用这种行为操作主体可以将所有风险都留给外界环境，而自身则在规则的保护下立于不败之地。在相对短的时间内物质世界的实物资产与虚拟经济资产在总量上并不会产生太大波动，在视为稳定之时这里的收益由其他所有人来承担。
