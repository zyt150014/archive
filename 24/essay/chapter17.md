# chapter17 视角

# 第一人称，第二人称，第三人称是非常典型的不同视角。
# 在状态变化的过程中，第一人称视角是最直接显然的思考角度。在第一人称下，我们的感受器能直接感知现实环境的变化，也能直接基于这种感受进行相应的实践活动并得到直接的反馈。而除了第一人称视角，我们有时也需要从其他的角度进行思考。
# 而当你开始采用不同角度时，这里就会存在一个问题，也就是庄子与惠子游于濠梁之上时发生的情况。庄子从鱼的角度思考，认为自己在水中悠闲自在。惠子则觉得庄子不是鱼，又怎么会知道鱼会怎么思考呢？庄子则以同样的思考逻辑反问惠子，惠子也不是庄子，同样也不会知道庄子的思考和感受。而在最后庄子则利用语言的二义性，巧妙地回避了惠子接下来的不同视角分析。从这个故事我们可以看出，从主体的角度思考是自然而然的，之后切换视角进行的思考在将主体的思考方式带入客体时也会受到主体思考方式的制约。在庄子代入鱼的角度时，想到的或许是免除了人所会遇到的烦恼，增添了游于水中的快乐，但却无法实际感知作为鱼的不便之处，以及可能在鱼眼中对人在空间中自由行动的期望。在切换视角后我们也应当对于我们的思考方式做出改变，但这里可能是永远无法到达的现实，因为你的所有观测得出来的结果与实际之间的差别在你的观测之外。
# 这个现象也可以用另一种方式来表示：你无法确定周围的人是类似游戏中的npc还是与你一样能够以思考的个体。在你眼中，我可能并不能进行独立思考，也只是管理者想让你看到的样子。既然在本质上无法证实也无法证伪，那么一直思考这件事情也很难获得新的结论。在表象上，我们可以认为大家都有不同的视角，并以这个视角作为思考角度。
# 在从个人的视角转向他人的视角时，我们对自身的行为往往会进行自我美化，从而扭曲事实。在黑泽明执导的罗生门电影下，盗贼，武士的妻子，以及被杀的武士在同一个现实结果中从各自的视角给出了美化自身形象的论述。荒山上被杀的武士，在各不一致的语义表达中成为了被迷雾笼罩的惨案之中最为真实的叙述。
# 这也是人性复杂的一种体现。如果能深入了解人性，得出每个人做出论述的动机以及每个人视角的事件碎片，我们可以从头还原出事件的真相。但事情的真相也可能并不重要，最后的结果也已经确定，对此进行的思考分析相对而言有更多意义。
# 当你从他人视角来思考时，可能会使你陷入混乱。在和其他人之间进行交流时，为了表示礼貌个人一般会顺应对方的习惯。在两种互相冲突的习惯同时存在时，这种自我规训就失去了支撑。过多的他人视角思考会削弱个人对自身的影响，引出邯郸学步的笑话。在个体之间的交流时，以他人的视角来确定自己接下来应该做什么，结束交流之前应当如何做出反应，也是一种身处其中时已经意识到了的流程。
# 在表象基础上构建的他人视角思考，也存在其积极意义。在个人的视角看来，自身的行进方向在一定范围内一直都是向前的，而从侧面来看则可能在逐渐偏离目标的方向。当局者迷，旁观者清。从更大的范围视角来重新审视现状，往往会有新的感受。
# 这种更大的范围也就是以客观事物的角度来进行思考。这种思考和先前将自身经验带入他人的视角不同，理论上这里不存在主体的概念，是将状态变化的整个过程当作进行思考活动的背景，这也就是绝对的理性。但是个人的思考方式和个人之间有着紧密的关联，从个人引出的在另一个角度的思考也总是会带有个人的特点。但好在，我们可以在尽量了解事情全貌时做出更加理性的认知判断。
# 在回到个人视角之前，我们在客观角度的思考也在影响着我们自身。在客观角度的思考结果上，我们可以向上建立各种人称在不同视角的思考。各种不同的思考来源于不同的底层路径，互相之间又纵横交错。对所有的不同人称的视角所进行的思考结果进行分析，在你推演它们的状态走向时，你对世界的认知也在这个过程中逐渐完整。
# 视角是跳出个人所在范围的工具，也服务于个人与所在范围的关系。
