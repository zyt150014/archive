# chapter4 预测

# 当我们了解简单系统特定范围内的所有信息，我们就可以在时间线上进行预测。
# 这里的预测指的是沿着时间线向前的方向，还未发生的认知判断。如果在现在的时间点，我们可以使用当前的所有数据，我们也可以推测出历史。所有的文字，音像记载，地质活动的残留痕迹，人类进行实践活动而改造世界的项目，还包括所有个人的记忆，都是推断过去不可或缺的资料。而在进行推断之后，真实的历史也可能变形，所以我们能接触到的推断资料也仅供参考。
# 向前的预测相比则更加可靠。一方面来说，即使是变形的历史也具有现实意义，而“人类从历史中吸取到的教训就是人类从来不会从历史中吸取教训”，这也是某些事件一而再，再而三发生的原因。另一方面，当下的数据与我们的距离更近，我们对这些数据的使用更加熟悉，这些数据相比历史数据对未来的预测结果有更大的贡献，也更加直接了当。这种预测也具有更多的现实意义。
# 我们最熟悉的预测之一可能是对于天气的预测。在靠天吃饭的农业时代，今年的收成往往需要看天公的脸色。而在不能直接改变天气环境的时代，个人往往会参考天气来进行生产活动。什么时候适合播种，什么时候适合收获，都需要在天气这个大前提下再结合个人的情况进行调整。当下的天气预报，已经可以大致模拟几天之内的天气变化，这也是指导个人生活的重要工具。
# 我也经常会进行一些设想：如果存在具有足够大算力的计算工具，有足够多的能源能够用于计算，有足够多的空间存放运算结果，然后使用原始的数据获取方法将世界的元数据导入到这台计算机器，我们能了解到无穷远的未来吗？如果不能，那我们能推断出形成这种数据的过去吗？
# 然而我们既没有足够的算力支持，也没有足够多的能源。当前能够为人所利用的能源绝大部分来自这个恒星系向外释放能量的一小部分，而对核能的利用也才刚刚起步。世界的元数据我们也尚未找到有效方法可以获取，这些数据以概率分布的形式在一定范围内取值。我们能观测并进行实践过程的，一直都是模糊的世界。
# 所以如果我们能实现跨越，成为更高等级的文明，那在这个过程中，我们如何把握住大的方向？如何拨开前进的迷雾？
# 不同的时代各有各的问题。“以经济建设为中心”在过去的很长一段时间，发挥出非常明显的效果。这不仅改善了生活，也增强了人民信心，形成了一种正反馈调节的机制。将其作为参考的话，我倾向于需要先实现能源的广泛，完全，以及深入利用。对能源的完全使用加强了获取能源的效率，能源获取效率的提升让我们可以有更多的能源可以使用，这也是一种“物质财富极大丰富”。
# 时代不断在发展，我们对未来的窥探也只是冰山一角。亚马逊雨林的一只蝴蝶扇动翅膀，也有可能引起北美大陆的一场飓风。这虽然不是完全正确，但这可以揭示一个结论：起始条件的微小扰动，最终的结果可能完全不同。失之毫厘谬以千里，某种程度上世界是一种混沌系统，我们能做的是预测趋势，并随时修正。
# 另一种预测也很重要，我们也经常用到。当和你交谈的对方说出一句话之后，你很有可能能够猜中对方接下来会说什么；又或者当某个电影情节发展到一个阶段，很多暗示都让你能够设想出故事的结果。这些语义上的预测是概念空间的延伸。不同的个人有不同的关注点，而且使用的概念空间也存在区别，所以在语义上的预测也往往不同。但是如果抛弃个人的概念空间之间的区别，转而使用公共的概率空间，那么注意力就是你所需要的。这也是语言模型的一种浅显的原理。
# 在上一段中使用的“但是如果抛弃个人的概念空间之间的区别”这句话，再结合之前我们所讨论的关于认知的内容，很容易预测到接下来的一句话的内容“转而使用公共的概率空间”。我想这种即时的体现也能让我们构建出逻辑自洽的个人概念空间，从而对公共的概念空间产生积极影响。那在这里我也可以提出一个问题：在语言模型，多模态模型之后，下一个是什么？
# 语义的预测对个人来说是有意义的，但这并不能完全代替现实的物质。而语义的二义性也是我们不得不面对的一个问题，这种二义性导致了以此为来源的语言模型的同样的问题，也是个人概念空间以及公共概念空间与现实的不同本质。那么实现语义的准确性很显然是我们的下一个目标。
# 预测未来，推断过去，我们在时间线上旅行。
