# chapter40 货币

# 在经济生活中，作为信用点的外在表现，货币在公共概念空间中的价值由不同区域的政府部门来保证。
# 这种概念来源已久。以贝壳作为一般等价物来进行物质交换的时间点，在具体意义上可以被称为商品的事物也比较少，大部分都来自于生理需要所进行的捕猎与采摘活动。而事物总在不断变化，化学元素作为化学层面组成物质的基本元素，而这种层面在个人的生产生活方面也具有特定的含义，在选择了金这种性质稳定，在获取上有一定难度，在个人生存的地球上的总体数量在一个比较好的范围内的化学元素作为公共概念空间中的一般等价物之后，货币的外在表现也走进了下一个阶段。对于黄金的共识在长久的影响中流传到了当下，在政府形式的区域经济状况下产生的不同国家的法定货币在新的体系下组成了一种复杂的结构。
# 这里首先就存在一个问题，如何控制这种法定货币的发行量？在一个特定区域内的所有物质交换，在流通效率的要求下客观上就需要一定的货币。这里的在市场上流通的货币也占据了发行单一货币的主体所制造的所有货币的其中的一部分，另一部分不在市面上流通的货币则不会影响流通的货币与商品之间的兑换比例。当然这二者之间也在不断地互相转换之中，在整体上保持平稳的状态。
# 这也只是一种较为常见的状态，我们也当然可以设想出其他的可能状态。如果发行的所有货币都进入了流通市场，那么很容易推断出来的就是这里必然会发生通货膨胀，同样数量的货币其购买力发生了显著的下降。这种简单的问题作为货币的发行者显然了解，而增加银行这种现金储备主体的存款利息也是一种显然的手段。在整体的经济运作过程中，这一环的变化也就会在每一个环节的传导中影响整个流程。加息的手段可以促使持有现金的主体更多地截留经手自身的货币，在更大的可能性中减少整体上的货币流通，维持经济生活的良好运转。
# 但这里也并不是一劳永逸的解决方案。在运转良好的经济生活中，生产效率的提升使得同一时间的商品数量也得到了提升，这也要求着更多的货币供应。那我们可以再想一下先前所说的利息的内在逻辑。银行在向个人货币提供额外利息的同时，也暗示着特定数量的货币实际价值的下降。这里是很容易理解的一点：银行自身并不参与实际商品的生产与消费，其职能在于在支付利息的条件下从社会上获得相应的货币，并使用这些货币进行宏观层面的投资，保证投资回报率超过所支付的利息即可在经济生活中获胜。这里也就导致了个人货币价值的下降，当然也有其他的因素，我也多次说明运转的环境是所有因素组成的整体，没有什么能够完全脱离。
# 所以将剩余货币存在银行虽然在分析中会导致价值的下降，但这种方式对于认知不足的个人来说也减慢了这种下滑的速度，在策略上算是中策，对于这方面的认知价值也在个人参与经济活动的过程中得以体现。好的一方面是经济可以向前发展，更大的蛋糕能让增长不那么多的群体也从中获益，相比于零和博弈来说也具有更多的积极意义。不好之处在于这种流通在不同层次的个人身上所造成的风险有着不同的影响力，八二定律在差距的扩大之时也可能会加速这个差距的扩大过程，而这显然并不是正确的方向。
# 如果在一个经济社会中的一部分群体掌握了超过半数的货币资产，那么这个群体在同一意志下的选择也就是这个经济社会的走向。当然我们每个人在经济活动中的参与都有着一定的影响，而在众多的影响因素下不需要超过半数即可达到同样的效果。利用体量的差距将自身的货币在一定时间内转变为更多的货币，并且保证这一定的时间足够短足以让总的货币价值上升即是金融游戏的打开方式。
# 在这种乐此不疲的过程中，空转产生的效益最终总会经由经济机器的运转到达每一个个人，这也是发行足够多的货币如果没有足够的经济发展来支撑就会逃离蓄水池进入市场，而这也并不是一件好事。
# 货币在交易中让自身的价值得到了认可。
