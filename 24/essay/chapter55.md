# chapter55 音色

# 在区分不同类型的声音之时，音色是一种判断方法。
# 频率在声音层面所产生的整体效果，也就包括了音色的属性。使用工具将听觉信息转变为视觉信息后，我们就可以进行更多的分析过程。首先我们可以对声音的产生进行一些了解。事物本身的状态在一直改变，在微观层面上也在进行无休止的运动。这种振动信息经由传输媒介被个人的接受器所接收以进行后续的处理，和色彩一样形成了大脑层面的具体感知。在个人的感受器所能产生反应的范围之内，也就是与可见光类似概念的可以听到的声音。
# 不同的事物有着不同的内部构造，也就有着不同的振动形式。在相对静止的状态时，事物自身微观层面所产生的运动所产生的振动信息在没有处于个人所能接收到的频率的声音的范围内之时在宏观层面也就可以视为完全静止。但所有的状态都在不断改变，所有的振动信息频率在足够的信息量的情况下覆盖个人范围的声音种类也足够多，在整个生命长度的范围内也无法穷尽。
# 在个人所能听到的声音的范围内，频率的具体体现也就是音调。同样的一种声音波形，使用不同的音高来进行呈现，这也很容易区分这两种情况，相似的走向也是一样。响度作为另一种主观感受，主要受所接收的频率信息的多少影响。距离声源越近，所接收到的同一种信息也相对更多，在整体上所带来的感受也就是响度的增大。而音色作为描述声源本质信息的属性，具有一些不同的意义。取多个频率信息之一来进行分析，个人能区分波形变化的最小区间的升降在单个频率数据中的所有数据变化也就是音色所指的具体内容。
# 基于不同的内部构造，声源处所产生的声音也就有了不同的音色。在受到外界的不同影响下，再现过程的之间的差异也就以音调为主。如果这种影响足够强大导致内部构造产生了变形，那么也就形成了不同的音色。声音传播的介质也相当于一种加工过程，同一种原声在传播介质的影响下也会附带上相关的信息。这种效应很明显的一种体现就是自身所听到的声音和使用记录工具进行记录之后的再现过程，个人在不同的感知中所进行的同样呈现也来自于自身的结构信息。
# 在所有可能的音色中，个人的主观感受赋予了不同的价值。无规律的噪声与有规律的声音演出在声音层面并没有本质的不同，但在某个时间点个人与声源的相对位置确定时，一定的声调，响度与音色组合也就形成了个人对于这种声音的全部感受。对于声音的空间感知与色彩相似，不同的接受器产生的感受之间的细微差别在大脑中进行处理后增添了相应的方位信息。
# 除了无规律的噪声与规律的声音演出，普通的各种形式的声音才是我们接触的常态。处在清醒状态的个人在面对各种各样的接触所发出的振动信息时无法停止这种接收过程，但可以选择进行选择性过滤。个人自身的关注列表中存在着具有更多意义的特定音色的声音，这些声音与个人也相对有着更多的关系。从最初接触声音之时，其他人的声音成为了最熟悉的音色之一。这种声音所表示的语言含义在语言系统尚未建立之时无法了解具体的语义，但是声音的音调，响度与音色属性已经可以形成相应的感知。推测不同的属性所表示的不同语义，并且在感到不安之时听到熟悉的声音所形成的安全感也是事物发展过程中的选择。
# 使用声音作为交流的工具，在传递振动信息的目的上也达到了要求。但这种传递方式在广播的特点下所有可以接收到信息的个人都能有所感知，我们有时可能并不需要这种特点，而希望能过滤出特定的接收对象。将所有参与人员聚集起来，然后进行最小范围的声音活动，是最直接的操作方法。点对点之间的交流也有现实的场景，这主要由视觉数据来进行传递过程。如果我们已经了解了这里的起点与终点，并且在全局下具有唯一性，那么这也就是一个特定的过程，所有的点对点通信在整体上也组成了全部的交流通路。
