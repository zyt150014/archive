# chapter15 积累

# 个人经历的所有状态，这也就是你的积累。
# 这种状态的积累无时无刻不在发生，这也是其他积累的基础。个人的状态积累也就是个人概念空间的构建和完善，为个人后续的发展提供了各种选择机会。
# 这种积累也可以被称为经验积累或者知识积累。就像长期记忆和短期记忆一样，这种积累的作用时间存在一定范围。随着个人不断使用某种积累，并逐渐对其他的积累变得不熟悉，个人的积累状态也在不断变化。而在个人的发展过程中，个人也逐渐习惯于使用某些积累。
#　个人在使用经验积累后，可以逐渐获得对应的报酬，也就是物质积累。对于一个白手起家的人来说，整个人生的过程就是在积攒经验，获得经验积累然后使用经验积累获得各种物质积累。
# 这里其实就存在一个问题。客观条件上每个人的经验积累和知识积累都不同，以当前的积累去进行实践产生新的积累的速度也不同。这种不同也是一种比较重要的区别，同时也是学习能力的一种体现。而从经验积累和知识积累，转变成物质积累的过程在大体上是线性关系。个人可以依靠经验分析以及对物质世界的了解，在某种物品价格波动之时低买高卖获得时间价值，或者是从事单位时间价值更高的事情。
# 而从另一个角度看，物质积累真的就是物质积累吗？
# 每个人都是由当下组成你的物质所组成的。这些物质在微观上和其他的物质没有本质区别，在化学元素层面就只有一百多种。但是这些物质之间不同的组合关系，在个人层面形成了能思考和行动的个体。个人的物质积累，在人类群体范畴内的含义也就是声明了对其他不能思考和行动的个体的所有权。这个所有权也只在与他人的交流中生效，也就是在个人层面，其他层面上不会生效。
# 当你在人类群体范畴内进行交易活动时，你也就是在将积累凭证转化成现实的物品。当你购买了一件衣服时，在微观看来你拥有了一段时间的组成这件衣服的物质的所有权，而在宏观看来你也有同样的时间的使用权。在这件衣服上会发生的各种过程你可以进行干预，但无法避免。随着物质的状态发生改变，时间逐渐流逝，这件衣服也会变得越来越旧。在你外出时，与各种灰尘和微生物的接触也是自然而然的。在你进行干预之后，也就是进行各种清洗工作之后，你延长了这件衣服的使用权。
# 当然你也可能并没有获得更长时间的使用权，也就是在进行干预之后你可能将其一直闲置下来，而在一个不知道什么时候的另一天直接放弃使用权而已。现在其实我们就可以稍微计算一下当下我们我们已经拥有使用权的物品的时间范畴。将积累凭证的总量和时间区间做一下除法，也就能得出单位时间所需要的凭证数量，也就是使用成本。这也可能是我们相对关注比较少的地方。
#　另一个角度，个人对不能行动和思考个体的所有权也可能会被随时剥夺。在现实的物质空间中，所有物理子的选择并不因个人的意志发生变化，最常见的也就是自然灾害导致的财产损失。无论是台风暴雨，还是地震海啸，这些发生频率较高的现象也往往都会让部分人失去拥有的部分物品的使用权。另一方面则是可能是在个人主观上或者客观上某个物品变成了无主之物，也就是个人的遗忘和他人的侵占这两种情况互相作用。
# 这样看来，物质积累是个人顺应社会的运行机制而声明并受到社会承认的权力，知识积累则更多地依赖个人的意愿和精神状态。知识的更新和增长使个人的概念空间更加完善，也更加具有自我解释性。同时这种积累在个人实现物质的积累方面也发挥着作用，个人从而拥有了更多使用权。
# 但我们也可以更加地了解这两种积累。知识积累是各种不同概念之间的联系的建立和消失，所有的概念都在客观上存在，这种积累可以看作是所有联系可能之中的一种而已。物质积累也是在时间和空间上将组成你的所有物理子与物品之间建立联系，作为主体的你和作为客体的物品的这些联系也只是组合关系的一种。
# 苏轼也在赤壁赋中就明白这种联系。无论是江上清风，还是山间明月，我们都只是在建立不同概念之间的联系。
# 积累过程也一直跟随个人，复杂的组合关系也是如此。
