# chapter37 内在含义

# 在艺术的创作中，事物的内在含义有自身独特的意义。
# 首先我们可以讨论一下前文所叙述的表现形式的内在。在某种全新的艺术形式被创作出来之后，组成这种艺术形式的内在就逐渐获得了特定的含义。由外在所承载的内在我们也可以进行一些划分，大致可以分为主体情感以及情感之间的关联与相互作用。无论是以艺术为基础的悲剧还是喜剧，这些具有情感内核的艺术都有着自身独特的情感关联方式。
# 我们又可以对这些关联方式进行数学分析。这种关联方式也和先前所提到的很多概念一样，可以看作是一种有向图。这个图中的不同的点包含了不同占比的各种情感，同样也具有相对性的多少的区别。图中的边则是在双向交流中传递着各种信息，同时也体现了不同点之间的联系的强弱。
# 这种简单的内在含义分析是一种朴素的思想，在当下的设定下我们也可以这样认为。对于莎士比亚所创作的悲剧作品《哈姆雷特》，作家本人则有着自己的评价：“一千个人的眼中有一千个哈姆雷特。”仁者见仁智者见智是一种被普遍接受的观点，但是《哈姆雷特》所要表达的主体思想却比较明显：在文艺复兴浪潮中思想的解放与对于螺旋式上升的发展中的社会的写照。
# 在艺术作为一个整体所包含的内在含义与外在表现随着状态的变化过程而不断发展之时，在整体上形成的固定印象反映了内在含义的特定意义。对于某种特定的艺术，在我们将其划分到某个范畴之时这既是使用了这种艺术采用的事物所蕴含的独特意义的结果也是他人对于这一艺术范畴的起因认知。当我们谈论喜剧时，我们会将具有夸张手法，具有引人发笑的目的性的台词以及公共概念空间中的美好结局等要素组成的艺术划分到这一个范畴，而在先前的喜剧的了解之上当个人接触到具有相同要素的艺术之时也成为了这种艺术被称为喜剧的原因。
# 在整体上的对应方式同样也适用于组成整体的每个元素。在个人创作某个方向的艺术的过程中，采用一些具有典型意义的事物是一种常见的方法。在一段特定的艺术片段中如果我们想要表达厄运和恐惧之时，在天空中的一群乌鸦就可以渲染出不详的氛围。以乌鸦代表厄运也有一些特定的原因：1.成群的乌鸦往往在黄昏时出现在坟头等处，这在对于黑暗有着原始的畏惧的人类看来有着恐惧的本能;2.乌鸦作为食腐动物，其现身之处也往往意味着生命的终结，而事物从高潮到结束的这种向下的发展也让人感到悲凉；3.在个人驻足查看或者是快步离开时伴随的乌鸦的凄苦的叫声，也在听觉上传递了不幸的信息。尽管当下当我们遇到乌鸦之时公共概念空间中的情感概率分布以厄运居多，但在不同地区和不同的时间点乌鸦也可以被当作积极方面的情感的表达载体。
# 这种事物所包含的独特意义既是个人的联想作用，又是对先前表达方式的一种沿用。这种独特意义作为一种逐渐形成的惯例，在个人的不断使用中既是一种意义的强化，也让这种内在含义，也就是特定的情感比例以及不同情感比例所组成的元素之间的联系形成了个人的审美疲劳。
# 在我们再次审视内在含义所包含的基本元素时，我们也很容易就能了解到内在含义的发展方向。在这一层中，我们使用不同的情感原料，按照不同的占比进行调配，并在与其他情感组合体进行相对大小比较以及使用不同的强弱联系构建出了整体的艺术。情感原料在客观的自然环境与个人的生理条件下基本不会发生变化，所以情感的混合以及不同的组合结构才是内在含义的唯一不同点。
# 使用这些要素组成的艺术，在客观中就已经占据了自身所在的意义空间。后续个人与社会整体的解读，则是在主体表达脉络上的游离结果。这种游离结果在个人的独特性的影响之下也具有独特性，在状态变化的过程中我们可以拥有不同的内在含义体验。83%喜悦、9%的厌恶、6%惊恐、2%的愤怒是对于蒙娜丽莎的微笑的一种情感占比分析，复杂的结构可以组成无穷的内在含义。
# 内在含义在形成独特意义与产生全新的意义之间往复。
