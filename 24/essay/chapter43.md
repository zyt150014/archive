# chapter43 外部交易

# 在讨论了内部之间的经济之后，视线可以转向不同货币主体之间的交易。
# 与内部的经济活动类似，外部交易也由需求所主导。物质世界资源的分布不均，不同群体在不同产业上面的优势，也都成为了造成这种需求的原因。作为现代工业的运转血液，石油这种原料的加工制造出了各种中间产物，后续就是进一步的加工制作过程。而作为信息技术产业发展的引领者，在制定各种标准以及提供各种信息服务之时也成为了覆盖全球的产业。
# 这种交易遇到的第一个困难就是不同货币之间的兑换比例。在当下的众多货币中，特定货币兑换其他货币的汇率在一定的时间范围内会保持稳定。如果政府放弃自身的信用，在内部造成的经济问题也会经由汇率在外部体现出来。这种信心建立起来需要足够的主体重视，偏离这种信心则很容易。
# 我们可以继续讨论一下普遍的交易过程。在主体a与主体b之间的经济活动中，a首先需要b提供的制作原料，所以主体a之内的个体支付盈余或者主体进行负债来提供相应的货币，从而交换到b提供的原料。a接着对原料进行加工制作，转变成各种产品进行出口，b则使用先前从a处获取的货币来进行交换，主体a最终又回收了货币。当然这里也存在a提供原料b进行加工制作的过程，这种流通过程也可以扩散到所有主体。这里无论是a使用的货币还是b使用的货币，都不一定是自身所发行的货币，在一般的情况下会使用第三方锚定了更高信用的货币，制作原料也只是从开采到成为商品的中间产物。
# 在经过了一定时间的外部交易之后，不同的主体会产生不同的结果。某些主体在付出更多的劳动价值中积累了足够的盈余，在统计所有进出口之时其出口收入大于进口支出，在外部交易中出于顺差。既然存在顺差，那也必然存在一些主体在外部交易中处于逆差。在不断地积累中，这种顺差与逆差也需要进行干预来实现动态平衡。在逆差与顺差的主体中，哪一方面占据了优势？
# 拥有逆差的主体的逆差数值在实际上也就是内部客体的盈余减少量，但是却并没有减少主体的负债。在盈余减少之际，自身内部的经济活动会受到明显的影响，这也会推动主体产生更多的负债来让内部客体获得更多盈余，但这是有代价的。外部获得其盈余的主体也在使用获得的盈余来兑换逆差主体所提供的商品和服务，货币数量与经济体量不匹配的后果在所有内部客体承担之时，国家的运转也会出现问题。而如果主体透支自身信任不承认外部盈余之时，也就让自身脱离了世界发展的发动机。
# 在逆差主体不愿放弃个人信用之时，外部的顺差主体则可以再顺差的兑现过程中扩大自身的影响力，在外部交易中处于优势地位。
# 逆差主体当然也明白这些事情，关税作为重要的限制手段也具有明显的收效。在每笔交易中附加足够的赋税，也就能大幅减少盈余外流的速度，但这也会推动商品价格的升高，内在客体在没有备用选择之时也只能接受更高的支出。这样看来，这种增加关税的手段更像是伤敌一千，自损八百的平庸策略。
# 这里还有一种另外的出路。逆差主体在利用自身产业优势之时可以生产出外部顺差主体所需要的高附加值产品，在资产交流中取得优势。这种形式在获得了大量低附加值的廉价资产的同时只付出了少量的高附加值资产，在外部交易中处于顶尖地位。不过这也不足以高枕无忧，如果这里的顺差主体不存在对应的需求那么这条道路也会产生堵塞。在长久的外部交易中，丰富自身的产业并增强自身经济实力才是以不变应万变的高效手段。
# 伴随着不同主体之间进行各种合作越来越紧密的趋势，良好的外部交易环境是所有交易参与者的共同利益。某些个体在鹬蚌相争渔翁得利的短视利益的驱使下的具体行为，也都在所有主体的视野之内。使用威压维持的外在表现，也在暗中标记好了自身的价格。
# 外部交易加速了经济发展的进程。
