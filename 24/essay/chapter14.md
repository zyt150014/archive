# chapter14 价值

# 一件物品被人需要，解决了相应的问题，便增加了价值。
#　当我们谈论价值时，随之而来的往往是价格。对这二者的讨论也已经有很多了，价格是价值的表现，价值决定价格等。对于同一个物品，其市场价格会在一个合理的范围内波动，这也是合理的价格范围。一般而言，在自由市场中，随着供应不足导致的价格上升，在提高了利润的同时也会促进更多的供应，从而使价格回归。但这里也存在其他许多种情况。在价格上升后，在能找到相对低的替代品时人们会减少对这种产品的消费，然后根据个人需要，转而消费各种各样的替代品，这种情况下这种产品的消费和生产都会减少，然后它的价格会在新的价格范围内波动；另一方面，如果这种需要不是非常强烈，人们也有可能直接减少甚至完全去掉个人的这种需要，从而导致价格的下降。
# 价格的产生源于组成物品的各部分的价值以及各部分之间的组合价值。人们使用各种开采工具获得的各种生产原料，其价格基本上都由各部分的价值组成，这种价值受开采难度，丰富度，物理属性等各方面的因素影响，并且各部分之间的组合关系也没有价值。在工业化之后将这些原料加工成各种各样的工业产品，就改变了这些原料的组合方式，在整体上增加了不同部分之间的组合价值。如果是将工业制造的各种零件再组合起来，也就是在更高的层面增加了组合价值。
# 这也就是物品的本身价值。在不考虑特定的需要时，同样的物品在同样的时间点具有同样的本身价值。这些价值客观存在，而我们如何降低获得这些价值的成本？
# 扩大规模往往是一种既简单又有效的办法。从生产原料开始，就可以选择丰富度高的区域进行开采，来降低开采成本。将这些开采出来的原料运送到工厂又产生了运输成本，这里也可以综合考虑运输消耗时间和运输单价等因素计算出总体成本最低的运输方式。将材料运到工厂后，在批量生产中将加工成本平摊到每个时间点的每个物品上。后续物品从工厂到分散的个体之间的运输成本，则在空间上无法避免。在这种情况下，从整个环节来看，需要找到最适合的工厂的物理空间位置以最小化原料到工厂的运输成本和物品从工厂到个体的运输成本。
# 为了获得更高的收益率，也可以提高物品各部分之间的组合价值。最常见的就是工业制造的各种复杂的机器和人工制造的各种艺术品。假如我现在已经有了一批零件，这批零件存在成本也能以相应的价格售出获得收益，则可以计算出对应的收益率。而在我有了将零件加工的机器的蓝图以及找到代加工工厂之后，我可以重新计算出总体成本和总体的收益率，以及这个过程中的风险。在二者之间进行对比之后，我可以选择收益更高的选项。
# 个人的需要是价值实现的最后一步。如果一件物品不被人需要，人们不能使用它来改善自身状态或者提高效率，那价值就无法转变成使用价值。这里的个人需要也非常复杂，包括个人作为整体的一部分的特定属性，也有和其他部分区别开来的情感偏向。作为整体的一部分，我们首先具有生存需要，所以能满足生存需要的物品对人类整体都具有普遍价值；另一方面，个人的情感偏向会提升或者降低对于某种特定物品的价值判断，各种文学艺术作品就是这种情况广泛出现的场合。
# 最终价值以价格的形式被量化，并指导后续的同类产品价格。在客观条件下，一件产品的价格在时间线上必然是不断波动的，这也是众多影响条件在某一刻的输出结果。这里就有了一种新的方向，在时间线上的低买高卖从而获得时间价值。
# 这也是各种投机行为的本质。在对未来的价格预测或者是个人的认知判断下，人们在进行各种投机交易。但个人的投机交易也是众多影响条件中的一种，未来的确定要到打开薛定谔的猫的盒子之时才能知晓。
# 另一种时间价值则是个人的主观时间价值。在同样的一天的8个小时中，如果你选择完成某些事情，那么这种时间价值就是这些事情的价值。在这些事情中，每个事情的单位时间价值也不一样，这里客观上存在一种具有最高时间价值的事情。如果将其他低时间价值的事情替换成这种高时间价值的事情，那整体上就能获得额外的时间价值。但这些事情之间也往往存在逻辑关联，一件事情的完成依赖于其他事情的完成。这样看来，更好的方案可能是完全替换成平均时间价值高的事情。
#　价值是物质的一种本质属性。
