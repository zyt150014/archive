# chapter46 斗争

# 不同的主体在某件具体事件上的利益冲突或者是观点分歧，然后在主动性下利用客观事物的行为，在物质空间中的相对位置的争夺，也就是斗争的含义。
# 使用参与者的不同类别来进行区分，我们可以将斗争分为纯粹的内部斗争以及内部与外部的斗争。这二者存在一定关联的同时也可以并行发生，在具体的历史环境下的主要矛盾决定了后续的发展路线。
# 这也正好可以用最近的一些具体斗争作为例子。最后的封建社会在革命浪潮之下分崩离析之时，社会在这种巨大变化下处于无序与混乱的状态。在内部的自我探索过程中，出现了各个方面的力量。不同的力量在来源于不同的主体之时也就有了斗争与合作的可能性，然而走向却往往是斗争。处于外部的主体在自身的需求之下也可以参与到这些斗争之中，这也加剧了形式的混乱。
# 在对内部的利益的争夺过程中，有着内部认同的个体会遇到内部革命与外部侵略双线并进的局面。在分析这两种事情的轻重缓急之后，先放下内部的斗争过程，从而集中力量进行外部斗争，是维护自身长久利益的选择。而在足够多的具有内部认同的个体的行动下，也就建立了与外部力量进行斗争的统一战线。
# 所以这种统一战线也会随着外部力量在斗争过程中的失败而走向瓦解的结局，这也是解决矛盾的客观规律。在对当下的主要矛盾有了认知之后，主要矛盾的解决会将更次一级的矛盾转变为主要矛盾。在接下来的内部斗争中，总会以代表了更多个体的一方的获胜作为结束。这个过程也必然艰辛并和外部斗争一样以流血作为具体的代价，从而顺应同样的历史规律。
# 在完成了这些程度上更加激烈的斗争之后也就是以改革这种小范围的斗争作为主要的研究对象。先前我们也已经提及了改革及相关的概念，而在内部与外部的温和斗争也以政治关系作为一种外在表现。这些斗争在上层的思维交流中以下层的军事行动作为实际表现，这也是接下来的主题。
# 构成内部的军事力量的个人来源于内部，也需要对自身存在内部认同。在进行相关的审查之后，这些个人也需要在长时间的服从中获取相应的信任。这里也存在原则问题，满足了原则之后的个人能力对于主体才具有相应的使用价值。从更大的整体来看，如果所有个人都能在同一种运行规则上达成一致，那么这些政治关系和军事行为也就在不重要的结果中仅存参考的意义。在不同的视角中，这种一致性却难以保证。在做出最坏的打算之时，不同的主体也都选择了这条政治关系的道路。
# 在不同的主体为后续可能存在的斗争进行准备之时，兵役成为了一种具有成效的手段。在和平年代，组成国家的主要劳动力也应当主要从事各种生产生活活动。在具有一定的重视之后维持军队人数在总体人数的比例，并将其作为国家运转的基本条件。作为补充的兵役，让更大一部分人在了解了军事行为的具体执行方式与行使手段之后，在成为了主要劳动力的同时也可以随时满足军事需要，从而构建一个更加合理的体系。
# 在大大小小的各种斗争中，参与斗争的各方在没有明显的策略上的差别之下其所参与斗争的数量级也就成为了这场斗争的关键因素。在足够大的力量对比面前，其余的因素也会变得微不足道，但在差别不大的情况下，微妙的决策变化所造成的斗争走向也无法预料。在先前以少胜多的案例中，天时，地利，人和作为最常归因的具体概念，也说明了形成这种结果的原因。
# 让我们再次看待一下斗争的含义。在利用客观事物作为所有的斗争方式，我们所要实现的目标是位置的争夺。在斗争一方对于客观事物有了更深的演变认知之后，也就能将万事万物作为自身进行斗争的武器，而不需要个人利用主动性去利用客观事物的这个过程，这也就是运筹帷幄而决胜于千里之外的基本条件。
# 如果各种政治关系和上层的思维交流足以达成双方都满意的结果，那么这种两败俱伤的流血斗争并不是好的处理方式。在维持自身的谨慎看待之时，我们也本应在共同的命运下拥有一个更好的未来。
# 斗争行为不是原本的目的。
