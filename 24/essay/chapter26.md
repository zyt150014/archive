# chapter26 科学

# 科学通常指自然科学，这也是状态改变过程中遵守的规则。
# 我们对于科学的认知以及遵守科学的规则进行实践活动由来已久，这个过程也在潜移默化中进行。科学是一种实践行为，依赖于在实际的物质世界中进行相关的实验最后得出的结果。这个条件有一个基本的前提，也就是在物质世界的不同的相对位置，对于相同的科学实验应当有着相同的结果。但是在理论上我们不可能找到相对位置不同但是外界环境却完全相同的实验环境，所以这个地方只需要保证在一定范围内在进行科学研究的层面上可以视为相同即可。对于水在室温下是流体的形态，且会慢慢蒸发的科学来说，这个室温的范围也足够大，在普遍的情况下我们都能得到这个科学实验的结果。
# 但是，如果对于相似的科学实验，我们观测到了不同的实验结果，这可能只是实验条件不同的原因，但也许会有一些科学上的新发现。在科学历史上，我们有众多的意外收获。无论是居里夫人发现的镭元素，还是解救了众多生命，治疗细菌感染的青霉素，都是在研究某些具体的科学之时得到的意外的成果。当然很多情况下也都是个人在进行实验时不够严谨或者纯粹是实验过程被某些条件干扰但是个人没有意识到而做出的错误的认知判断，所以我们也需要首先对自身的实验进行完整的检查。
# 如果根据个人判断得出了不可能的结果，在当下的物质世界中只有可能是个人的原因，客观上运行的物质世界并不会出现逻辑上的漏洞。这里的假设是并不存在一个管理员类似的身份来设定好物质世界的所有参数，而直到现在我们也尚未发现发现由于管理员的疏忽而能被我们发现的矛盾之处。在三体中，对理论物理有着强烈信仰的杨冬，在进行的各种科学实验的结果之间逻辑上存在错误，但是杨冬却对实验环境的一致深信不疑。作为读者我们当然知道这是三体人在这些实验中造成了影响导致了实验结果之间的差别，但在杨冬认知之外的力量的影响下让她的所有科学实验都失去了意义，一切也都只是偶然而已。
# 作者的这种想象也是按照自身对于科学的认知所形成的自由意志的思考，这种行为在概率性上我也觉得是可能发生的。如果这里真的存在我们认知之外的力量，并且这种力量改变了我们的科学实验结果，那我想这种力量也大概率足以改变我们的记忆，让我们完全失去关于这个科学实验的所有了解。
# 在这种情况下，我们没有任何办法，而从个人开始进行认知和实践过程之时也就说明了这一点。而这种力量是否存在，无法证实，也无法证伪，这也不在科学的范畴之内。
# 科学最重要的一点就是可证伪性，如果我们找不到一个反例，那在找到反例之前我们就可以直接认为这种科学是被证实的。在个人对于物质世界认知不足之时，天圆地方是古人的所有想象，在没有了解现代科学之时将其作为被证实的科学并依赖它进行后续的认知和实践活动没有任何问题。而后来了解了太阳，月球，逐渐形成了地心说，后来又形成了日心说，而在现代科学之下这些理论都是站不住脚的，当下我们认为一切都在不断地变化，所有天体都在互相影响中不断地进行状态改变。
# 在对各种科学实验进行研究的过程中，了解到的互相印证的科学结论也容易引发个人对于哲学的思考，而科学与哲学之间也有着内在的联系。无论是建立了力学物理体系的牛顿，还是提出了相对论从而在尺度上重新建立起物理规则的爱因斯坦，在越来越多的科学研究的结论的积累下，越来越感觉世界的存在并不是一个偶然，在冥冥之中有着一种神秘的力量。所以科学的尽头是神学吗？而在科学与神学之间的哲学，相对于神学在当下可能更有研究意义。
# 在现实的认知和实践过程中我们总结各种现象产生的规律，研究事物的发展历程，然后对结果进行总结分析从而形成被证实的科学。依靠这些科学，又能让我们去更好地去进行认知和实践活动。这种科学的现实意义也是我们所关注的概念。
# 科学引出技术，成为了第一生产力。
