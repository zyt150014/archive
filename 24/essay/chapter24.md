# chapter24 社会秩序

# 在社会秩序构建的自由之下，我们在此生活。
# 一般来说，社会秩序有两种表现形式。一种是较为强制性的，我们也需要遵守的，能保障权利的法律；另一种则是依赖公共概念空间中的普遍原则，并最终依赖个人的特点而不同的道德。社会秩序的维持，需要二者的共同作用。
# 与权力和权利的产生与形成有着密切关系的社会秩序，我们也可以先看看它的演变过程。在权力形成之后，从权力的使用者，使用权力，个人顺应权力的使用而进行相关的实践活动的这个链条的构建中，也就逐渐成为了一种社会秩序。从这种最简单的社会秩序出发，我们在沟通方式完善之后在公共概念空间中设立了各种规则来保障个人的权利。在对应的违反的情形下我们也设计了一些惩罚，用来保证权力的权威性。
# 随着这种规则条目的越来越多，我们开始将描述不同方面的规则进行区分。不同的规则则由权力所指定的个人来进行管理，也逐渐形成了具有层次的管理体系。在中央之下，有分管不同事务的细分部门。这些细分部门又对地方进行管理，然后向下直到每个个人。
# 而在状态改变之后，社会秩序也需要进行一些改进来更好地满足状态的需求。社会秩序的不断完善，在达到了一定的程度之后我们也就可以形成一个比较稳定的社会秩序的版本，这也就是各种法律确定下来的第一版内容。后续的修正案，是对基本版本不适应时代的地方的一些改进。
# 这种强制的社会秩序，也是社会运转的基础。如果没有了这种社会秩序，我们也就没有了在进行认知和实践活动的范围意识。各种生产生活活动也会陷入停滞，社会也不再发展。而这对于所有的个人来说都是不利的，个人无法获得规模效应带来的收益，生活质量也只会下降。
# 在这种情况下，每个个人都应该会去遵从这种社会秩序。在中央，因为参与的个人更多，社会的发展更迅速，所以一些破坏社会秩序的行为也会得到很快的处理。但是在地方上，如果客观上没有人了解到个人破坏社会秩序的行为，那么也无从解决。对于一个能解决问题的人来说，最大的问题就是如何去发现问题。
# 在社会秩序的规定中，也包含许多禁止事项，而这些禁止的事项很多都能够带来足够的利益来让人去破坏社会秩序。这些事项大多都是以损害他人的权利来让自身获得更多的利益，也是我们之前所说在权利总和相对稳定的前提下所发生的情况。而这种的解决办法也只能去加大成本，让个人在获得这些破坏秩序的收益的同时付出更大的代价，这种加大成本的方法也来自于个人所拥有的自由的思考。
# 而比较宽松的社会秩序就是个人的道德。对于个人的道德，在公共的概念空间中也并没有详细的规则和具体的要求，这也就是完全依赖于我们个人的自我要求。但在一个模拟的社会中，保持一定的道德水平也是个人持续发展的重要条件。在最开始，模拟的社会中有高道德水平的人，也有中等水平和低等水平的人，当然这里的水平都是相对的。在个人与中高道德水平的人之间进行交流活动时，这种个人的特点也会逐渐加强这种交流活动，与低道德水平的人之间的交流活动则相反。当然你可能会觉得低道德水平的人因为有相似的特征所以也会互相加强联系，但更多的人会去偏向于和具有中高道德水平的人进行交流活动。在逐代的发展中，在整体上个人一般也具有一定的道德水平。
# 除了这种外在的因素影响，个人的内在因素也有相应的作用。个人在与他人的交流活动中也会有美化自身的倾向，这也让个人对于自己的外在形象有一定的追求。这种个人对自身形象的要求也能让个人发挥主观能动性去改善自身，从而更加符合多数人对于自身的期待。而相应的，这种更高的期待也对应着更高的道德水平。而造成这种美化自身的现象的原因是什么呢？
# 我个人认为这源于我们对于存在于我们的想象和思考中的以我们的标准来说的更好的自己的向往。一般而言，个人对于自身的各种缺点的了解会超过其他所有人，这也让我们去设想一个更好的自己有了明确的方向。更好的自己与现实的自己的差距，也就是美化效应的作用区间。
# 社会秩序在每个人的认知实践活动中得以运转。
