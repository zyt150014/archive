# chapter9 决定论

# 事情的发生是确定的，命运在所有选项中做出了选择。
# 从最本质的角度来说，决定论是两个不同物质空间状态之间的有向变化路径。我们通常将方向的起点称为起因，方向的终点称为结果，决定论也就是对起因和结果以及这二者之间的变化过程的思考。
# 我们可以先看下这个问题：基于现在“物理子”的所有状态结果，我们作为“物理子”的组合体，对于“物理子”的状态改变有一种原始的感受。而在我们不断地往前推导这种因果过程，我们能得到最初的原因吗？
# 我们可能比较熟悉“宇宙大爆炸”这个假说。这个假说首先确定了所有结果发生的原因，也就是所有物理过程都由奇点而来，而现在这种理论仍然只是我们的一种猜测。如果我们在概念空间中接受了这种解释方法，那让我们继续推导一下。在这个过程中我们会去使用一些当下被认为是真的原理，现在我们可以进行回溯。其一：当下空间中物理子的密度有限。其二：从无空间概念的奇点发展到现在这个时间点，空间的大小也是有限的。其三：在这个过程中不会出现物理子或者物理子的其他形态凭空产生和凭空消失的现象。综合以上原理，在奇点这个状态存在有限多的“物理子”，体积上是无穷小，导致密度无穷大。在推导到奇点之后，可以再往前试着推导。在奇点之前还有另一种形式存在的奇点吗？奇点也有可能是由已经经历过奇点膨胀而产生的另外的物理空间收缩而成，而做出这种设想则意味着这之间经历的次数我们无从得知。
# 可以继续采用物质空间的抽象概念思考方法。假设有最初始的状态，那么之后所有的状态都是从这个初始状态发展而来。在这个发展过程中，所有的“物理子”所能存在的状态是有限的，这一点也很好理解。如果从最初始的状态到当下的这一个阶段中，产生了各种各样之前不存在的状态，那么我们可以将其加入状态集。这种加入过程在存在最初始的状态的前提下是有限的，也就是世界无论向哪个方向发展，都无法跳出状态集。
# 你可能会觉得这个地方不太合理，因为这里将尚未到达的状态也加入了状态集。如果有着无限的未来，那么这种思考就不太合理，但是这里的无限可能和普通意义上的无限存在一定的差别。我们也可以抽象出一个所有状态不再发生变化的结束状态，在到达这个状态之前我们也不知道需要经历多长时间，“物理子”会发生多少组合变化。但是如果某种状态会发生，那么这也必然是所有“物理子”在这一状态的前一状态做出的选择。
# 考虑一下诺顿穹顶。我们可以想象一个光滑的拱顶，拱顶的最上方的质点可以达到某种平衡状态。在普通的受力分析中，这种平衡状态就会一直持续下去。但是如果存在一个微小的扰动，那么这个平衡就会被打破，这也是在实际中经常发生的情况。而使用数学模型，诺顿构建了一种穹顶。在这种穹顶中，如果我们将一个小球放在拱顶最上方的质点，那么按照先前的讨论小球将会保持平衡。而在计算了在只受重力的下滑过程中，会有多个解存在。这里的解会出现与问题毫不相关的时间参数，暗示小球会在任意时间内可能不想呆在穹顶就自发下滑。
# 那么按照上面看来，它似乎是非决定论，因为无法决定任意时间到底是多长。但是如果思考一下，上面过程中“微小的扰动”是状态发生改变的中间阶段。在这个中间状态之前这个系统保持平衡状态，也是所有的“物理子”的选择。而所有“物理子”在到达下一个改变的状态中，在微观层面上使拱顶小球速度的高阶导数发生了突变。这个地方的突变无法确定是多少阶，也无法确定方向和准确大小。在发生这种突变后，状态发生改变的信息从上到下传递下来，最终使小球从拱顶滑落。
# 这种解释看起来有点合理，但也无法解释这种突变为什么会产生。但是这是这个系统所有“物理子”的选择。
# 使用个人的自由意志来反驳这个观点的话，就陷入了逻辑漏洞。组成个人的所有物质也是“物理子”，在大脑中作为支配主体的“物理子”在某个状态决定了下一个状态的方向，而我在写完一个字的状态下决定了我继续写下一个字的状态。
