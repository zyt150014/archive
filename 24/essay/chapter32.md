# chapter32 自身

# 集中了物质交换和思维交流的主体，对自身产生兴趣也理所当然。
# 这种兴趣首先产生了对“我”这种概念的提问。我是谁？我从哪里来？我要到哪里去？柏拉图所提出的人生疑问也依然让我们产生全新的思索。对于哲学上的定义与如何的问题，我们可以进行一些简单的讨论。首先对于“我”的概念，已经有很多种解释了，一般认为是用来和他人进行区分的概念。他人与我二者互为补充，没有了他人的概念，我的概念也就失去了意义，反之亦然，不同的人称称谓也就是这种他人与我的概念的具体体现。我从自然中来，也要到自然中去。在使用一个具体的范围将我们都包括在内之后，我们可以进行缩小，直到在这个范围之内不包括任何不属于我的概念的物质，在这个范围外也不包括任何属于我的概念的物质，这一时刻我们就将自身与外界进行了完全的区分。随着状态的改变，我也在与外界进行着物质交换和思维交流，在客观上我的概念也是动态的。
# 在个人对于“我”的概念产生认知之后，自我认同也是一种重要的经历过程。在自我认同的不同阶段，也会有不同的呈现。一个没有自我主见，完全依赖他人看法而生活的人是自闭的，也没有任何形式的自我。从自闭状态也很容易进入迷茫与探索的阶段，个人开始去思考自身与社会的关系，也在这个过程中完善自我的概念。在度过这个阶段后，个人对于自我的认知也就达到了一个新的高度。个人会完全了解自身的需求，并评估自己的能力。在此之后，个人可以通过对自身的详细分析找到一条完全属于个人的道路，并在踏上这条道路之时坚定自身的想法，同时依赖自身去决定他人意见的重要程度。
# 无论是自我否定还是自我怀疑，都是走向自我认同的过程中的小插曲，被这些所困扰而不陷入其中则是一种重要的理性思考。在你未完成自身设定的目标时，或者是你对未来没有方向时，客观上的这些发生概率相当大的情形会让个人进入内耗状态。这种焦虑和迷茫的基调是一种向下的螺旋，在焦虑和迷茫的增长中伴随的是痛苦和恐惧的增长，同时也是认知和思考能力的下降。然而这里也并不存在一个超然的存在能够直接扭转当前的局面，唯有自身能实现自我救赎。
# 在自我认同让自身可以信服之后，自我实现也就是接下来的主题。自我认同产生了自身的完全认知，这既包括个人的当下现状，同时也有自身的周围环境的情况。在对这些信息有了详尽的了解之后，自我认同可以构造出一个符合实际的个人发展路线，自我实现则是对这条路线的遵守，也可以是更高的要求。社会的运转需要个人的自我实现，而设置的激励机制则是一种行之有效的方法。在个人一步一步到达了设定好的地方时，这里也已经没有了自我实现的存在。
# 但个人在认知与实践活动之外，留给自身的思考时间却并不多。而在这短暂的时间之内，这种思考的效率也不足以达到能够找到可以长时间指导个人路线的程度。过程往往是曲折的，但是纯粹的目标就在那里，选择一个长时间的目标与安排到达目标的所有细节路线这二者来说，前者则要容易得多，而且通往广场的路也不止一条。
# 在这里我们也可以讨论一下具有自由与独立思考能力的主体对于自身的研究与思考方式的理解。首先在物质世界中，我们可以肯定物质的存在早于思考的形成，所以我们可以找到我们的思考过程的起始点。第一步：物质组成了可以认知与思考的组合体；第二步：这些组合体接受外界信息形成概念认知；第三步：这些概念认知扩散到自身，形成闭环。在这三步过程中，每一步都存在着信息的丢失与理解的偏差，在第三步获得的原始数据在经过一圈的处理之后回到自身与实际的情况也并不完全一致。这种不完全一致在最后的综合分析过程中会产生负面的影响，所以具有自由与独立思考能力的主体并不能完全理解自身的研究与思考的生效方式。而在这个结论之下，这种个体制造结构比自身更加复杂的复制体也几乎不太可能，但这里也有其他的办法，使用规模更大的简单复制体在整体上能够达到规模较小的但结构更加复杂的个体的效用。
# 这种方向的理论基础也已经分析过了，在物质世界中也有许多先例。遵从这种规律，社会也能走上更高的发展水平。
# 自身既是自我认同，也是更大整体的部分。
