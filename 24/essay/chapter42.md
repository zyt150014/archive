# chapter42 盈余

# 当有人进行负债，那也就产生了对应的盈余。
# 我们可以继续讨论货币在经济生活中的作用。从最开始的发行货币环节，铸币权则由执法主体唯一指定。虚假货币的制作在个人的行为中似乎是利用货币的传导性窃取了来源于整体的价值，但同样制作人在这个过程中也付出了相应的劳动。不可否认的是，这种行为不论是出于个人兴趣还是经由利益驱使，对于经济的发展也只会产生消极作用。
# 获得了铸币权的主体成为了所有货币的来源。与执法主体的联合效果，在保证这些货币的有效之时从相应的执法客体获取对应的实物资产或者虚拟资产，这个过程就让主体产生负债并让客体产生了盈余。但是铸币的成本也是我们无法忽视的价值流失，这里也显然需要保证铸币成本相比于实际制作出的货币来说占据较小的价值。在纸张作为廉价并且制作技术成熟之后，再结合防伪工具，即可进行大规模的投入使用。使用金属制作而成的硬币，在制作成本的提升之后也逐渐失去了锻造的价值。
# 继续看一下后续的流程。在主体产生负债之后，形成的盈余则在市场上进行流通。在这种流通中，个人使用劳动产生特定价值来进行交换，而这个过程并不会让盈余增多，却同时实现了个人需求的满足以及驱动个人进行劳动从而让庞大的国家机器持续运转，这也体现了货币的内在含义。这里也可以多思考一下，来分析最初的盈余。在货币进入市场的这个过程之中，客体在整体上需要首先进行主体所需要的劳动来获取货币实物，主体也达到了短期的具体目的。
# 但这里好像缺少了一个关键环节。税收作为主体调控经济市场的手段，一般看来是主体的支出来源，但这种复杂系统也没有这么简单。如果所有经济活动都没有了税收，那么会发生什么？
# 在主体的持续性的需求中，自身的负债也水涨船高。在理想模型中，每个人都获得了越来越多的盈余，如果没有足够的经济增长，这种盈余带来的则是系统的崩溃。但因为个人对盈余的使用具有时效性，这也有利于蓄集更多的盈余。这里也可以举一个简单的例子：如果货币发行主体所对应的客体在同一时间都将自身在银行的所有资产都进行取出操作，这就产生了一次低层次系统的瘫痪现象。
# 税收的存在同样是一种金融工具。主体使用税收来回收自身所发行的货币，这种手段的使用方式也可以根据发行的多少进行调整，让经济在发展过程中保持平稳。
# 所以我们可以了解到，市场中总会存在盈余，这种盈余在经济的流动中既从主体获得，后续又被主体回收。而将注意点放到主体身上之后，我们会发现我们忘记了考虑主体的目的。在集中了所有客体的劳动可能之后，主体足以在规模化的分工合作中规划整体的发展路线，在宏观上取得经济的可持续增长的成果，而这种成果也应由整体所共享。
# 而事情往往不会这么理想，分配的权利也并不完美。无论是数据的虚假制作还是外在表现的良好包装，主体在内部的分歧中也会造成判断的失误。如果本身就存在分歧，那么可能连经济的可持续循环都会存在问题。
# 所以在这种可能不算最优解的经济系统中，个人可以有怎样的应对手段？这里我们需要再次提及先前去中心化范畴内的概念，也就是逃离中心化思想衍生的产物。使用这种产物作为可信共识的外在表现，需要完全客观的规律来保证其不被篡改。而如果客观的数据产生了冲突我们应当如何处理？首先这里应当只有一种连续的数据，在最有可能的概率中我们选择了少数服从多数的思想。
# 在去中心化的产物中我们如何计算准确盈余？在参与这种实践活动的个人处在一个有限的范围内时，这种产物与主体所发行的货币之间也就很容易产生波动，被这种特性所吸引的投机者的目的和最开始的动机也并不一致，这种消极意义也会让我们产生疑问。
# 建立在私有财产不可侵犯的基础上构建出的现代经济系统，使用负债和盈余来驱动运转，在很长一段时间内也难以抛弃这套体系。
# 盈余不是真正的资产。
