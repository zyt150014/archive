# chapter39 经济

# 社会需要物质的生产与流通，价值的交换也让经济得以发展。
# 在先前谈论价值之时对于经济我们也已经有所涉及，而这个概念本身也在个人的行为中得以产生。价值的生产与消费，也反映了经济的强度。而具有内在价值的事物，在客观上可以分为实际的物质与不存在实物的特殊商品。这里所说的实物所指的也只是实物在特定时间段内的使用权，不存在实物的特殊商品则包括数据服务以及个人的服务，这实际上也就是不同艺术的特殊结构与在物质世界中改变物质状态的体验。
# 首先来讨论一下实物产生的经济。在产业上，传统的农业以及工业就是产生实物进行物质交换从而发展经济。从生产上来看，这二者并没有什么区别。工业相比于农业的先进之处在于，可以进行成规模地提高生产效率，并且规模的提升也几乎不会导致生产效率的下降，而在大多数情况下在良好的组织结构的优化下能获得更高的效率。在蒸汽机引导发展起来的第一次工业革命，推动了社会发生了巨大的变化。相对于传统的手工纺织，新的纺织机的出现让个人付出更少的劳动即可获得大量的劳动成果，这也加速了财富汇聚到少数人手里的过程。后续的第二次工业革命也属于工业范畴，围绕着电力的产生既是对于化石能源的一种替代，同时也为第三次科技革命提供了动力支持。计算机作为第二次工业革命和第三次科技革命的中间人，在集成了各种工业产物之时采用电力开始驱动了数据结构的改变所引导的数据服务。
# 再来讨论一下虚拟的商品。在产业上，服务业也就是以无实物从而提供感官以及情绪体验从而形成经济。服务业的起源也很简单，最开始来自于劳动的替代。个人在物质世界中进行认知和实践过程之时，将某个棘手的工作或者是因为一些原因而未能实行的过程转而由他人代理，并在这个过程中进行劳动的交换或者是债务的声明，也是经济发展的重要支撑。这种改变物质状态的体验随着实物产生的经济的发展也发展出了另一种数据的服务。这种数据服务可以看作是物质分布的结构数据，在不同的结构变化中产生了自身的经济价值。
# 既然以是否承载于实物的使用权我们可以划分出两种经济形式，那么这两种形式之间又是一种什么关系？
# 在个人需求的满足方面，生理需求和安全需求多半由实物的承载所实现。这既包括个人基本生存所需要的生理所需，也有在基本生存边界之上的比较体面的选项。而更高层次的情感需求则更多取决于对于不同艺术的体验感知，以及他人所提供的个人服务。自我实现需求则是二者的结合，鱼和熊掌皆是个人所希望得到的，这里的二者也并不存在完全的对立关系。实物经济具有稳定的特征，在一定的范围内并不会具有很大的波动，因为客观的使用权已经锚定了对于人的使用价值，而这种价值本身是一种普适的概念。而由服务业衍生而出的偏向虚拟的经济则具有较大的波动性，在不同的个人眼里的价值也千差万别。这种特性也正好满足投资的基本逻辑，待价而沽从而获取经济的成果。
# 而个人在经济的发展中可以做些什么？在先前我有一种朴素的思想：个人在庞大经济机器的运转中承担一些具体的劳动工作，并在公共概念空间中的信用机制下获取相应的信用点。同时个人也在消耗整体经济的运转产物，并以之前所获得的信用点作为简单的交换，这种分工在追求效率的方向上成为了最好的选择。但这里不存在完美的方式，个人在分工行为下的工具化是个人实现需求的阻碍。
# 所以我们需要清楚地明白：经济的原始发展来源于个人与整体的需求，后续的效率提升也是为了满足需求的变化，以实现相互的匹配。所有的这些经济活动都需要以人作为主体，而不是在个人需求概念上起到相反的作用。当然我们也可以考虑其他的影响因素，但是作为思考的主体其自身与其他的客体本身就存在不同，这种差别对于不同层次的思考主体也同样适用。
# 经济的发展让个人的需求得以满足。
