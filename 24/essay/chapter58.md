# chapter58 不可知

# 将不同视角构成的层次概念以及整体的所有信息作为认知的来源集合，个人在已知与未知这两种情况下在主观上也就有了一种划分方法。
# 个人所拥有的时间有限，单位时间的效率也有限，那么最终形成的已知边界也有限。观察一下边界的外在表现，与边界相邻的外界则是具体方面更加深入的领域。在特定的方向上，个人很难去突破前人所达到的范围的并集，在这个方向上的最远的已知信息也几乎无人知晓。这个并集在当下也更可能是更高层次的一小部分，也存在更多的未知信息等待我们去探索。
# 在客观上，所有的可知概念是确定的。如果某个概念可知，那么在主观上可以再次进行已知与未知的区分。如果某个概念不可知，那么我们也就无从得知这个概念。这种不可知的概念之间其实也有一点细微的差别，受当下科技限制而不可知的概念与完全无法了解的概念在当下没有进行区分的意义，但在状态变化中也会逐渐变化。
# 思考一下熟悉的可知概念。既然这种概念可知，那么个人就可以在习得过程中形成自身的认知。面对实践过程中所遇到的各种问题，在公共概念空间中可能存在相应的解决方案也可能没有。如果个人对于这个方向缺乏足够的认识，那么也就不可能采用这种解决方案。基础信息的缺失会导致后续的问题，所以这也提醒我们来扩展自身的视野。而作为一个棘手的问题因而缺乏应对策略之时，这可能也就是一类重要问题。这类重要问题的解决可以让一系列将这个问题作为下位层次的整体得以增加相应的进度，进度满了之后也就让这件工作带来的效益成倍提升。
# 再来思考一下不可知概念。当一个人不了解具体的事情时，那么也就失去了评价的资格。将所有不可知范畴内的元素看作一个不可知的整体，这个整体上的部分信息也是相对可知的。我们知道它是不可知的，我们知道这里的不可知可以分为完全的不可知与条件限制下的不可知，我们也知道这种不可知是客观存在的。除了这些少量信息之外，我们一无所知。
# 用当下已知的概念进行组合，我们就能得到许多不可知的概念。使用足够的数量级进行围观或者是宏观陈述，这就是在条件限制下的不可知。而如果是完全的不可知，那么我也无法给出一个例子。比较接近这种概念的可能是在决定论中最小粒子所做出的选择，我们无法预测，在足够长的时间之后会如何发展则不可知。
# 这就像物质在自然发展中的运动一样，在层次之下具有一定操作空间的客观事物会有着粗浅的可知信息，也就是连续时间点所处的大致相对位置以及周围环境。完全的数据信息除了可能以假想形式存在的管理者已知外也无人知晓，而这也是事物的一种本质属性。
# 个人在将未知转化为已知的过程中，如何选择是一件重要的事情。与个人可知的所有事物相比，已知只占一小部分。基于个人的不同能力也有不同的策略，来更好地符合个人需求。对于一些简单的问题如果个人已经非常熟悉，那么继续进行的重复活动也是无趣的。过于困难的问题在远超个人能力的同时也只会带来挫败的感觉，也不太适合。在个人角度上既有挑战性并且在思考后能解决的问题能激发出个人的兴趣，并在循序渐进中得以发展。了解了这一点后，对应的利用方式也逐渐形成。
# 在某一个时间点的个人，所有的已知来源于下部层次之间的组合过程。随着组成个人的物质之间的联系不断变化，得到的已知范围也不一样。在漫长的时间过程中，物质在客观层面的不变性保留了部分信息，个人的已知也转化为所有物质的信息组合。在遥远的未来会发生什么是当下组成个人的所有物质所不可知的，但这些物质也会经历所有的变化过程。初始数据的叠加过程不会影响先前已经存在的数据，在某种意义上这也增加了广义上的我的意义的已知集合。在情感几乎不再变化之后，可能意义本身都不再存在内在的含义。但至少剩余的时间还属于我，在一切消逝而感到平静之后。
# 不可知来自基本逻辑。
