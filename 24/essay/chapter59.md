# chapter59 平凡的一天

# 这时正是深夜。
# 天色阴沉，乌黑的云在天空中翻滚涌动，星星点点的灯光见证了整个过程。微弱的风，在细微的波动中暗含气势，预示着后续的强烈展示。树枝上的树叶随之而动，摇曳声也一并产生。暗淡的月光照着空无一人的路，只有一片沉静的景象。
# 事物大多数都陷入睡梦，熬夜的人却保持清醒。过去的记忆涌上心头，既熟悉又陌生。无论是悲伤还是表现出来的其他情感，也无人在意。在一分一秒的流逝中的思考过程，让人怀疑能否平淡地活着。实际上每个人心里都明白，但保持沉默也造成了未知的可能。思考再三最终沉沉睡去，在无知中期待着光明，也梦想着与现实有差距的世界。
# 乌云在月光的背景下愈发乌黑，黑白相间的光纹展现着自身的震撼，闪电与雷声也与之相伴。要下雨了。
# 一下就要下个痛快。雨滴从天而降，清冷的风也掠过世间的所有事物。人们在天花板下消受着自然的幸运，树间的鸟在看向天际之时等待着自己微漠的悲哀。世界渐渐趋于明朗，天空也转向淡白的色彩。这是清晨的声音，校园中又兴起了生命的活动。人们也被轻松的空气所唤醒，一切都好像和往常一样。
# 就像简单的描述一样，读书声杂乱无章却让人感到熟悉。教室里的人们在自由自在地打闹，追逐着风的声音。透明的窗户折射着光，记录着操场上人们奔跑的身影。树叶上残留的雨水也不时落下，完成了下雨的整个流程。
# 下课铃如期而至，笔却不肯放开老师的手。楼下的道路熙熙攘攘，一段时间之后又回归平静。阳光继续倾斜，下午的到来又让空气中弥漫着声音。
# 太阳正在注视着这个世界。在阳光找不到的地方，有与人相处的阴影。老师在讲课，看着每一个偶然相遇的人，感叹这人世的迟钝。人们在或多或少的理解中看向黑板，不间断地做着笔记。午后的一丝疲惫的感觉，进而转向个人在放空中对身体的控制，是最令人高兴的时刻。
# 在教学楼屋顶上执行相应任务的飞机，发出的震动声不时打断了老师的讲课，下课之后找老师问一些问题也就有了原因。窗外，各种尘埃在光线中现出身影，在空气中沉浮。树间的鸟发出悦耳的叫声，随着空气的运动而传播开来。
# 不知不觉中时间逐渐流逝，转瞬即逝的暂时紧跟着过去的脚步。昏黄的日光显示着黄昏的到来，人们在最后的下课铃声之前也迫不及待地思考着自己的晚饭。铃声响起，动作迅速，学习过程被暂时中断。少有的闲暇时光也吸引了校外人员，操场上空飞舞的风筝在人们突兀与和谐的感觉中共存。
# 黑夜来临，熟悉的铃声也再次响起，标志着时间流逝的间隔。新的题目被老师写在黑板上，而窗外也有一位相同的老师写下同样的题目。灯光在不断地闪烁，人们也在眨着眼睛。
# 将孩子的晚饭带到学校的家长在完成了任务之后也略微逗留，在公开的成绩表前仔细查看，寻找着自己可能认识的孩子。道路两旁的路灯发出亮光，照亮了无数人曾走过的路。
# 黑夜就这样持续。路灯之下的树叶在地面上显示出来的斑驳的影子，与不同的人影重叠在一起。不期而至的铃声预示着一天的学习过程的结束，同学们纷纷离开学校。没有表现出任何留恋，每个人都认为自己不是最后一个。月光驱散了云层，重新放下这个世界。每个寂寞的星星都在孤独地燃烧，在天空中回忆着生辰。树间的鸟也早已进入自己的世界，无声地抗议着不时的喧闹。
# 反射阳光的月亮越来越刺眼，掩盖了星星的光芒。云沉默不语，安静地看待着这种变化。流动的风随心所欲，已无人在意它的行踪。黑夜不再黑，光明充满了整个世界。
# 一点一滴的时间提醒着人们，土地发出了叹息。道路掩埋着人们一个又一个的梦，无视着人们的回应。寂静处，强烈的感情淹没了人心。回顾中的潜移默化的事实摧毁着脆弱的心灵，以至于无法让人去忽视。可怜的执着只能换来一处心灵的缺口，失去了希望的事物也在不合时宜中发生了难以预测的举动。总有一天，世界会全然改变。
# 此时正是深夜。
