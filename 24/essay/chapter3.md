# chapter3 真与假

# 认知结果往往可以划分成对立的两面，也就是真的方面和假的方面。
# 同样也符合先前所说的个人概念空间和公共概念空间存在区别的情况，我们个人对于真和假这二者的判断条件并不存在完全确定的标准。但一般而言，我们会选择公共概念空间的普遍标准来对个人概念空间的判断标准进行调整，使个人的标准和普遍标准趋于一致，这也算得上是求同存异。
# 我们现在可以来思考一下：当产生新的认知时，我们如何确定这个认知的真假？
# 再次引用之前的话：“天空是蓝色的”。你可能现在会意识到下面几种情况：天空可能并不总是蓝色的，或者因为时间和空间的不同，天空会呈现另一种颜色，再有可能就是天空可能也无所谓颜色。你可以将另外个人认为的情况都列举下来，但是在普遍的观点下，我们会认同这句话，也就是认为这是真的。
# 至于假的情况，则非常多。想要将一种认知判断为真，一般需要一定程度的上下文支持，也就是这种认知所处的各种条件集合。在这里我们也可以迅速理解：将其所需的某个条件设置成与原先对立，我们马上就能得到一种假的认知判断。但这里还有一种情况，那就是即使设置成对立状态这种认知判断仍然为真。这种情况下我们其实可以注意到，这个条件很可能并不会影响真与假的认知判断逻辑，那么这个判断就不需要这个条件，那也就不是真假认知判断的所需逻辑，和最初的假设矛盾。
# 反证法在认知和实践过程中广泛存在。这种方法有效的原因是存在二元对立，并且这二元互为补集。二元对立意味着不存在重复，互为补集意味着不存在遗漏。可以看到方法的使用同样需要特定前置条件。
# 我们可以再讨论一下现实条件的真与假。对于某个确定的认知真假判断，其来源是个人和集体的概念空间；而集体的概念空间又依赖个人的概念空间而存在，所以这种判断的来源是个人的概念空间。但个人的概念空间又是由先前对于其他事物的认知真假判断而来，这里似乎存在循环依赖关系，也就是“先有鸡还是先有蛋”。
# 我们不需要去回答这个问题，所以我们可以先放到一边。现实世界总是不断变化的，这是我们的观测结果。以这个结果为基础，认知判断和个人概念空间的建立都在进行，这二者是互相依赖而又互相促进的关系。认知的判断需要经验累积，也会成为经验的一部分；经验的建立需要获得认知判断的结果，也会对后续的认知判断起指导作用。我们可以将二者之间的关系称为“共生”。
# 但这里仍然存在问题：个人使用一系列认知工具与世界进行交互，在这一层交互世界之下，还存在形成交互世界的另一种概念吗？我们真的是“缸中之脑”吗？
# 当我初次想到这个问题，我感到无所适从也无可奈何。这个问题也可以进行另一种解释：所有的认知，实践都是空中楼阁。当我们向下看去，是一片飘渺的云雾，而我们都处在云雾中。
# 当然我们本来也没有必要去关注这个问题，毕竟现在身处地球上的我不会去了解和关注多少光年外另一个星球上的演化轨迹。但这个问题却有一定的现实意义：在有限的时间内个人能接触到的边界是固定存在的，在这个信息茧房内即是个人的命运。边界之外的世界，对于个人来说无法逾越。
# 但在信息交流越来越发达的今天，这种情况也在不断改善，但虚假的信息也在增多，客观条件上我们也无法避免边界的存在。下一次当你需要进行认知的真假判断时，把握机会尽可能了解全貌，还是适配个人的需要，都是个人的选择。
# 真与假之间可能也并没有绝对的界限。在“忒修斯之船”中，换掉一块甲板的船是“忒修斯之船”吗？换掉多个甲板呢？将船上所有的零件都更换一遍，和用换下来的零件再次组装起来的船，哪一个才是“忒修斯之船”？
# 认为“现实世界是不断变化的”而随后建立的所有流动的个人概念空间和公共概念空间，在时间停止流动之时也唯一确定。这一刻的真与假，下一刻也可能是假与真。不断变化的世界使用时间这种度量工具比较变化的速度快慢，我们也无法逃脱。
# 真与假的认知判断，每个人也有不同的答案，现在你心中可能已经有了答案。
