# chapter8 物质空间

# 以具体物质形式作为承载载体，物质空间存在着多种层次。
# 在个人可直接认知的层次上，也是个人概念空间和公共概念空间的元素出现最多的层次。以这一层为中心，我们也在不断地向上和向下探索。
# 首先我们可以来分析一下物质空间。这里显然有一个作为研究前提的问题：存在最小的物质形式使所有形式的物理过程都在这个最小的层次之上发生吗？这个问题在引起人们思考之后，我们首先一步一步地向下进行研究。我们在不同层次，制造出各种各样的研究工具，用来拓宽已知的范围。而在进入分子，原子层面之后，再往下的探索似乎变得越来越难。在认为我们使用的工具是可信的基础上，得出的结果却越来越不可信，这里也是一种客观规律。和先前使用零件组装机器一样，在你使用一个零件去驱动另一个零件并进行嵌套使用时，每一层的误差都会成为下一层的影响因素，使用工具也是同样的道理。
# 而对物质结构的研究也好像是无穷无尽的。在个人这一层面上使用有向图向下能够到达的所有层次和研究工具，其中可能并没有能跨越到更下一层的通路。在当前的世界上下文下我们可以先简单地这样认为，毕竟现实无时无刻不在发展之中。而人如果使用抽象逻辑，我们可以首先确定一个递归出口，在个人的概念空间中打扫出一片场地用来推演。在物理形式过程可以发生的最小层次上，我们先使用“物理子”来作为别名。在一个封闭的空间中，如果所有物理子的位置状态各种参数都没有发声变化，我们就认为这里没有时间。这种状态可以一直存在，这里也用了时间衍生出来的概念来解释没有时间的情况，可以将其看作是既是无限，也是刹那，瞬间即是永恒。在所有物理子中如果任意一个的状态发生了改变，那就产生了时间概念。在这里会同时生成两个概念的有向箭头。其一是这种改变所使用的时间多少，其二是这种时间多少可以等效于何种改变。
# 在构建了抽象概念中的物质基础后，再往上一步步到达个人的概念空间这个层次上，我们无法得知这里有多少层次。但抽象总是有效的，从物理子到原子之间的过程，看起来是一片迷雾，那就将其当作黑盒处理。从原子，分子，高分子，再到我们熟知的现实概念，就可以在个人的概念空间中有比较准确的模拟，这是物质空间在用各种认知工具后在个人层面上的投影结果。
# 从这个微观世界的搭建过程中，我们可以了解到什么？
# 再从现实层面向下进行分析。同样是放在桌子上的一杯水，我们可以首先从它和我们的相对静止状态了解到受力平衡的原因，这里可以快速得出所受重力和桌面的支持力相互平衡。但在这个最简受力模型之下，可以进行更加复杂的分析。杯子和水在可以看作整体之外，又能对二者进行独立分析；所受重力也是对重力有贡献的所有“物理子”的计算求和，而支持力在我们放大观察之后也存在杯底中心和杯底边缘的区别。继续向下，我们先将杯子和杯中的水分别讨论。形成固体形态的杯子的分子之间排列紧密有序，之间存在强而有力的作用力；而形成液体形态的水的分子之间受力更加复杂，完全预测模拟结果也更加困难。而在继续向下之后，原子核和电子云的模型也是现代普遍用来解释这一层次现象的公共概念空间元素。
# 自底向上和自顶向下，也是二元分析沿着对立方向的体现。
# 再让我们向上进行探索。和微观角度的分析一样，我们也可以抽象出一个最高层次的概念，我们也可以使用“管理员”来作为别名。我们无从得知“管理员”的各种信息，也不知道“管理员”是否也会像我们一样思考。从“管理员”到可观测的宇宙，中间也是一片迷雾。在组成我们自身的物质能够有序地在足够远的地方出现时，我们会有新的思考。
# 在当前地球这个物质空间中，组成当前的你的所有“物理子”，在状态到达当前这个阶段之前，在现实层面会是各种各样的呈现方式。
# 物质空间作为唯物主义依赖的主体，也见证了所有的思考过程。
