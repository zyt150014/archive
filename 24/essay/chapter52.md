# chapter52 再现

# 将原本或者是其复制品在制作出来的时间点之后进行的执行操作，延长了先前的思考结构的生效区间。
# 再现需要结合复制的概念来谈。首先，我们需要原本的存在，这里的原本既可以来自客观的复制过程也可以来自主观的复制过程。具有可复制性的原本与经过复制后的副本在再现环节的一些方向上具有相同的效果，在关注这些方向的任务下使用复制过程所生成的副本即可。而这种再现过程，在存储介质随着状态的改变中也会逐渐丢失数据，因而再现的结果也会变得模糊。
# 除了需要原本或者副本之外，再现时机也需要一些方法对进行再现的对象的完整性进行检查，这里可以选择任意一种可以满足这种需求的方法。在一些情况下，我们想进行复制过程之时可能并不能获得对应的原本，使用副本再次进行复制活动是简单的处理方式。这种过程所得到的二次副本在同样的情况下也能再次作为被复制的对象，从而获得更多的复制品。在复制过程中可能的数据损失或者错误，会随着一步步的复制过程向下传递，每一级同样也有可能产生新的误差。对这一系列的操作对象接着使用再现工具来进行处理会得到不同清晰程度的认知印象。
# 记忆作为个人再现活动的主要执行客体，也是个人独特性的重要来源。当你在脑海中对特定对象进行搜寻时，个人概念空间中的再现也唯一确定了这种再现的接收者。由于所有的记忆都来自于先前所经历的状态改变过程，所以这种再现活动也是个人维持自身意义的操作方式。记忆所引导的再现现象，成为了个人情感的来源。
# 记忆在完全属于个人的优点下，也有在丢失后基本上无法找回的缺点。在牢记某些重要记忆的情况下，我们往往使用各种外部记录方式来将这种记忆以个人所能理解的方式来进行复制。个人所能理解的方式有两种含义，其一是指公共概念空间中的通用解释方法，其二是指个人在遇到这种情况时的加密解释方法。第一种情况所制作的副本能被大部分人所理解，个人一般也了解这个信息。第二种情况则与特定的个人具有比较紧密的联系，在不同的范围所形成的交集中也确定了唯一的理解主体。
# 外部的副本同样具有时效性。从最后的结果来看，所有的再现也是有限的过程，在具体次数的再现过程之后也就逐渐消失。这种认知在某个时间点也会被意识到，而对此我们会产生不同的反应。
# 在记录工具随着整体向上的发展也变得更加简单易用的同时，同一种再现过程所使用的数据也会变多。这里的数据指的是在个人理解层面上有用的数据，使用压缩方法来减少无效数据也能加速再现过程。这里的有效数据也同样会变多，在再现的主体逻辑没有发生比较大的变化时这里的主要贡献就是增加了细节的明确性。
# 讨论一下这里的再现类型。这一方面的再现所使用的第一版客观对象，在重现之时所展现出的整体结构就已经反映了这种再现的内在含义。这种整体结构所导致的情感变化，也已经足以达到自身目的的要求。在细节增多的过程中，这里的内在含义的展现也变得越来越清晰。从客观的角度，这种更加清晰的再现结果也优于先前版本的结果，但这里也有一个关键的因素。在状态改变的先后过程中，个人在特定方面的第一次再现过程和后续的相比具有自身的特殊意义。新形式的再现过程在增加了细节的情况下也丢失了先前的一些特征，在整体上的效果会带来情感变化强度的削弱。
# 在积累了众多的记忆复制品之后，这种重复的再现活动也会对个人制造新的复制产生影响。在固定长度的时间段内，再现活动的增多也就挤压了其余实践活动的时间长度。如果我们完全采用再现方式来填充剩余的时间，那这也就丢失了再现过程的本意。这不应该成为一种阻碍，而是一种思考对象。在可能性中我们寻找着新的思考结构，在细节中也存在原先的影子。
# 再现以复制作为运行基础，共同制造了认知原料。
