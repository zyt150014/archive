# chapter33 阶级

# 一旦有某些数据可以进行量化，那么就产生了阶级。
# 无论我们的经历如何相似，在某个状态对个人的所有相关数据进行量化分析，就可以将其归于某一类阶级。客观上的阶级不会消失，这也是每个人的唯一性体现。
# 而当我们谈论阶级时，最重要的两个阶级也就是无产阶级和资产阶级。在马克思的设想中，如果没有自身占有生产资料，那么个人即是无产阶级，资产阶级则是无产阶级的对立面。在对资产阶级社会的运作所进行的研究中，马克思发现了资本膨胀的方式，也就是在物质交换与思维交流的过程中获取剩余价值。但是仍有许多人并未明白这一点，所以伦敦东区最贫困的爱国者，只要一想起英国的工业和财富，也会不由自主地挺起胸膛。
# 而在这种资本的积累中，无产阶级与资产阶级所拥有的积累之间的差别也在不断扩大。尽管这种积累在物质世界看来只是一段时间的使用权，但在精心设计的社会秩序下这种使用权也会得到传承。不会流动的死水是没有未来的，这种不可持续的状态也会走向灭亡。
# 这里也不仅存在这两种阶级，每个阶级内部也有着更加细分的阶级。在对资产阶级的详细划分中，大资本家，中等资本家，小资本家是在对个人所占资本进行量化之后的多少作为标准来进行区分，而无产阶级因为不占有生产资料，也可以区分成为大资本家工作，为中等资本家工作，为小资本家工作，以及自身生产物资的农业从事者。我们可以不太去关注这方面的区别，但我们也无法忽视这种客观的物质世界中的差别。
# 我们可以思考一下无产阶级内部的细分阶级，来对整体的理解提供参考。数量最多的工业者，在缺乏某一个方面的技能之时能从事的工作比较有限，在很大程度上也没有进行个人选择的自由，在随大流之中为维持自身的基本生理需求而工作。如果个人具有一些技能，在资产阶级看来可以从个人的生产生活活动中获取可观的剩余价值，那么由于资本对于增值的需要，个人可以利用出卖的技能获得更多的回报，也能实现更多的个人需求。
# 这种简单的阶级分析也是阶级产生的原因之一。即使是在当下都没有某个方面技能的无产阶级，在客观上的不同认知能力与思考方式，最终也会产生不同的阶级。而无产阶级之中的普通中产，则沉迷于这种细分的阶级无法自拔。在向上的越来越窄的路径中，中产也正是千军万马过独木桥。在向下的注意中，也在尽力地维持自身阶级而不滑落。在由少量价值观所组成的价值评价体系中，这些无产阶级也无法跳出。
# 当然，如果对我自身进行阶级分析，我当然也会被划分到某个阶级。但我也并不过多关注这些，因为总有一些对你来说更加重要的事。我也更愿意从侧面去观察于思考，尽管这种观察和思考的结果或多或少会加入主体的个人色彩。
# 个人之间的阶级高低无法避免，在这种现状下对于我们的更好选择是存在一个比较合理的标准。我们需要尽可能地让更多的人在这个标准之上，让我们可以真正地从重复劳动中解放，从而专注于具有创造力的工作。
# 具有创造性的工作所产生的价值是难以量化的，各种文学与科学之上的新的发现创造就是一种体现。当下我们不可否认的是大多数人在前人的基础上也很难获得具有创造力的成果，这也是在从零开始的公共概念的构建过程中越发稠密的抽象概念。这个过程中影响最大的一点是尚未实现概念的继承，我们无法将一个成年人的所有了解转移到新生儿的头脑中让其以这些知识作为起点开始发展，这也让新生儿在到达具有同样水平的成年人的认知与思考逻辑中间不可避免地存在状态的变化。
# 而在先前所存在的已经产生区别的不同阶级，则可以通过外界的影响改变这个状态变化过程所经历的时间。这种优势在个体上的效用可能不太明显，但是在一定的范围内我们可以察觉到这种效果。具有更高阶级的前人使用这种阶级的优势所养育的后人在平均水平上高于较低的阶级，所以后一代的阶级变迁也多半来自于正态分布的边缘概率所导致的偏差。
# 我们被物质世界划分出阶级，我们也应当去减小阶级造成的影响。天地不仁，以万物为刍狗。但刍狗之间也能被量化而产生出阶级的概念。作为具有自由的独立思考的主体，从侧面的思考让我们能看得更加清楚。
# 阶级之间的对立化解，也是整体上的最优选择。
