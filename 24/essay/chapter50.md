# chapter50 工作

# 个人所做的所有实践活动，包含了工作的所有内容。
# 还是先对工作进行一下定义的描述。在个人利用能动性进行具体的行为过程时，也就处于工作与非工作的混沌状态。以时钟周期作为例子，将整个过程进行步骤的拆解，用于工作的时间片与两个工作的间隙时间片在交错中组成了连续的时间流。压缩这个过程中的非工作时间片的占比，或者是提高整个过程的持续时间，都能获得更长的工作时间的成果。
# 但在实际上，工作间隙中的非工作片段在进行部分削减之后，剩余的时间片在与先后的实际工作存在千丝万缕的联系之时不太能独立出来。我们当然可以根据实际的效果将具体的工作进行剪辑，然后进行一些拼接工作。但这种结果在不连续的间断中产生了逻辑上的问题，按照这种错误数据在后续的复现过程中会得到失败的结果。
# 通常情况下，具有明确需求的工作对于个人是费时费力的事情。分析一下具体的某个工作内容，在这个工作的作用下，将具体的事情的初始状态改变成了最终的状态。在记录了这个工作的完整流程之时，对于同样的状态改变要求就可以进行直接使用。但从工作的效用角度出发，完全相同的操作环境也比较少，大多数情况下都会进行一些微调。少部分的创新与长时间的维护过程是比较常见的工作流程。
# 与之前我们已经提到过的娱乐相比，工作与其的关系也并不是非黑即白。个人所从事的工作流程如果与个人的娱乐过程产生部分重叠，那么这也就在作为工作的同时也能拥有美好的感受。这种巧合在大多数人看来是一种个人的幸运，但可能只是所有概率性的唯一体现而已。这说明了工作与娱乐之间的某种联系，我们也可以继续思考：在利用个人的娱乐过程之时，能否在全局中有更多的概率性让个人的具体工作与其娱乐过程产生更多的重合部分？
# 在管理者的调度层面上，总体的投入不变的情况下能让最终的数值结果产生相对性的提升的话，这种功利性的调整在多个方面都能产生相应的益处。但这里个人所产生的机会成本与相对的提升比较起来也可能更高，而在权衡这二者之后，再加上个人在选择上的一定自主性，在动态过程中达到了一定的平衡。
# 所以个人与工作是一种什么关系？在个人尝试满足自身基本的生理需求时，将具体形式的劳动作为自身的工作从而在物质交换中维持组合体的有序程度。使用同一个时间段作为基准，个人在这段时间内所积攒的劳动价值在减去同一时间段内的消耗之时也存在一定的剩余。在这种剩余积累到一定程度之后，个人不需要工作也足以满足自身的所有需求，这也是财务自由的概念。这听起来很美好，但确实如此吗？
# 个人的思考与创造也是工作范畴的一部分。如果工作不再成为个人的一部分，那么我们也就失去了这种能力。在外界状态改变之时我们也就只有吃老本的这一条路，再加上沉迷娱乐的情况的话也就将可能的娱乐至死的结局确定了下来，而既没有工作也没有娱乐，那这种完全的无作为让个人与客观的事物之间也就没有了差别，这种结局也好不到哪里去。
# 在专注于当下的态度中，过于长远的观点没有市场。在个人的肉身消失之后事情的走向也与自身脱离了关系，不相关的概念之间也不需要互相考虑。在大量的样本中，多样性的选择是最有可能的情形。不同方向的尝试也能增加整体的经历数据，大流也已经足够使用。
# 与纯粹的娱乐相比，在结合了工作之后也就具有了一个更加均衡的环境。娱乐所制造的持续性的高强度刺激，在时间的效应下也会产生厌倦的感受。在结合了具体的真实状况之后，不断起伏的情感强度在参考了先前的所有资料所得出的结论后是个人层面上更有深度的记忆片段。
# 在短暂的过程中，我们也都应该去进行更多的不同的体验活动。而如果我们对此产生了一些疑问，并提出为什么，那么我会说因为管理者所构建的精妙就在那里。
# 工作需要拥有不被异化的劳动的内涵。
