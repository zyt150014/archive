# chapter31 思维交流

# 个人的概念空间之间的交流，是精神方面的重要活动。
# 在我们讨论这个时，我们也正是在进行思维交流。这种思维交流在当下以文字这种传递媒介形式而存在，交流的路径则是从个人的思考，到大脑的控制做出的行为，再到他人的感知器官的分析解释，最后在他人的大脑中形成思考。从思考的路径中我们也可以明确了解到直接的交流是不存在的，具有自由和独立的思考主体之间的思维交流也必然需要中间的传递媒介。
# 客观上的这种科学规律在我们的生活中也产生了许多问题，这些问题也来源于个人的输出和他人的输入存在着较大的差别，或者是个人的思维的思考方式与他人不太一样。个人的输入和输出与先前所说的概念空间存在着比较大的关系，我们对于同一个物质空间中的客体的在个人的概念空间中的不同映射概念或者是在物质空间中的不同的客体在个人的概念空间中却是同一个映射概念都会触发这个问题。当我们谈论某一天的某个时间点时，位于地球上不同时区的个人自然而然地会将这个时间带入到自身的时区，而解决这个问题我们可以使用通用的unix时间戳格式。在个人不能区分两种概念之间的细微区别时，我们也往往直接进行统称，无论是不同产地的特色番茄，还是被称为圣女果的小西红柿，在不了解这些概念之间的细微差别的个人来说在个人的概念空间中也都只是一种番茄的概念，当然也是一种西红柿的概念。
# 而如果两个具有自由而独立的思考能力的个人之间如果具有相似的概念空间，并且在利用概念空间中的元素进行演绎推理的思考活动也有着相似的路径时，那么这两个人的思维交流就很容易同频。伯牙和钟子期，作为思维交流同频的典范，是知音一词的重要诠释。在伯牙使用古琴进行演奏时，无论在心里想着高山还是流水，子期都能从演奏的乐曲中分辨出这种细微的差别，并了解到伯牙此时的思考活动。尽管伯牙和子期有着不同的视角，但这显然没有成为二者之间思维交流活动的阻碍。在子期因病身亡之后，伯牙的破琴绝弦也是一种再也难以找到知音的遗憾。
# 而我们也可以进行一下设想。这里可以存在完全直接的思维交流形式吗？
# 当我们思考这个时，我们对于自身的思考方式也并未完全理解。这里我也可以提出另一个问题：对于可以进行自我研究与思考的主体是否可以完全理解自身的研究与思考的生效方式？
# 在当下的语义环境中，我们的普遍认知是大脑在产生神经冲动的过程是思考的基本单位。在各种科幻小说中也存在直接使用这种在化学层面的电信号进行不同主体之间的思维交流的设想，并将其视为思维之间的直接交流。而从严格的角度来分析，对于直接的定义我们也无法实现直接交流。首先具有自由和独立的两个思考主体之间在物质世界中必然处于不同的相对位置，而在不同的相对位置之间也必然存在相对距离，在这个距离的变化中也会经过空间中的曲线。然后对于这种信号的解释也不存在一个完全相同的解码器，不同的自由独立思考的个体也有着不同的兴奋阈值。
# 在人为定义的通用交换协议的作用下，网络世界在超过个人的消息处理效率的情况下迅速发展。这种客观的人为设计不会被个人的需要而轻易改变，也是在整体的发展中所进行的合格的选择。而如果你觉得不好，你可以自己设计一种在各个层面都更好的方案，不论是容易理解，还是格式规范，或者是语义的明确，发展的含义也就是新事物对于旧事物的替代。
# 而思维交流的本质在广义上也属于一种物质交换，只不过这种物质交换所传递的不是能量而是信息。当然在之前所说的物质交换中光从太阳到达地球的过程中本身就是一种信息，而思维交流所承载的信息则源于个人的概念空间与公共的概念空间，是一种客观的物质世界信息的抽象显示。在个人对某种概念形成了认知的先前与之后，组成大脑的物质的种类与结构的细微变化也就是这种思维交流的结果。
# 思维交流也是物质交换之上的信息传递。
