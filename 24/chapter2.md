# chapter2 实践过程

# 依靠个人对现实环境的认知，我们可以来反向去分析一下个人对现实环境的实践过程。
# 在先前我们所讨论的认知中，当然也涉及到一些实践过程，比如以“_是_的”的格式，我们可以使用这种工具将某个特定的概念a进行一种比较完全的转换，使其形成另一个特定的概念b。因而我们可以对实践过程先入为主地产生某种自身的印象。
# 从整体的概念空间中，让我们考虑这种最简单的一次转换过程。这种过程非常直接，并没有什么可以去过多分析的地方，就像两个紧邻的神经元一样，没有更多的中间过程。我们可以将这些描述语言简化成两个字:"单转"。你当然也可以将其视为其他的别名，作为读者的你，这个地方也正是先前所说的个人概念空间和公共概念空间的交流体现。
# "单转"的实际生效场景很多，就像不假思索的思考过程一样，我们会认为这是自然而然的。被个体认为是自然的某种“单转”类型，在某个特定的意义层面上可以看作是强化的认知过程。在个人的概念空间中，不同概念之间有着或强或弱的逻辑关联。对于那些个人认为达到了某个阈值关联度的概念对，这里的关联程度会有更进一步的加强。这里是生物进化的结果，还是生物进化的原因？
# 当然，“单转”不可忽视地存在使用条件限制，而这种条件限制可能会达到一个比较高的数量级。思考一下以下问题：最高的动物是什么？这里首先加入了一个“最高”的限定词，对于处于“动物”这个范畴的概念在某个限定词的投影上进行对比。这里的动物对于不同的人也会产生不同的认知，这个地方为了简单我们可以采用公共概念空间下的认知。在经过个人的观测结果，多人观测结果的汇总统计，公共概念的生效逻辑之后，我们可以得出某个特定的动物a。这里的a没有现实的实体，是一次认知过程的抽象结果。但如果我们加上更多的限定词会发生什么？
# 比较明显的一点是，在加入了越来越多的限定词之后，我们可以将结果规划到任意具体的动物概念。这种形式的思考可能并没有很大的作用，毕竟个人的大脑运行的功率是有限的，并不能关注到任意多的限定词。另一方面，我们可以换一种方式来进行思考。
# 如果我们将加入限定词的方式产生的认知抽象结果的过程看作是两个相邻神经元的直接通路，那么这可能并不是生物的最佳选择，将这种认知抽象过程转换成多个“单转”类型我想是更加符合自然的规律。我们首先列出抽象过程的所有处理子过程，然后将每个子过程作为“单转”的实例，使用基本的处理过程来进行组合，从而实现“多转”。
# 个人对于世界的改造也是如此。在大量的普通的单次实践过程中，我们会不断强化某些“单转”流程，最终形成事实上的条件反射。这也就是“书读百遍，其义自现”的意义本质，也是个人将其组合成“多转”流程的基础。在改造过程中遇到的某些问题，我们首先使用多个“单转”模块进行整体骨干的搭建，这个过程中的思考需要进行深入，审慎地检查，也是个人大部分的思考过程所应用的地方。在经过了一段时间的权衡之后，最终总会提出一个个人的解决方案。在这个地方我们先不去讨论这个方案的好坏，而是关注这个方案的生成过程。每个个人的解决方案往往都带有个人的色彩，这也是个人在处理这个问题之前累积的概念空间联系的体现。
# 这种过程无时无刻不在发生，而个人也在这个过程中或者强化自己的“多转”流程，提高遇到类似问题的思考效率；也可以参考其他人的“单转”和“多转”流程，来判断在不同条件下的优劣以供选择。而从整体上来看，不论是个人的“单转”还是“多转”，因为特定的寻求更好的导向，往往会产生这样的结果：小部分人在投掷骰子时拥有更好的运气，这里的运气是主观条件和客观条件的综合结果，因而构造的“多转”流程具有更好的解释性和更高的效率；而大部分人的概率性并不存在什么特别之处，其中包含更多的公共概念空间的元素，所以这种构造“多转”的结果也并不出彩。而将上面的结果总结一下，我们容易得出这样的结论：在实践过程中往往是少数人扩展了整体的实践能力，大多数人会采用公共概念空间的普遍认知或者是学习少数人先进的实践方式来丰富自己的概念空间。
# 回到开头看一下，所有的这些分析都是依靠个人对现实环境的认知对自身实践过程的一种认识。这个地方是否涉及到自我指涉？如果存在的话这种递归的出口在哪里？
# 实践过程与认知密不可分。以认知为骨，实践过程为翼，在我们面前的是一幅广阔画卷。
