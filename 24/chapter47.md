# chapter47 竞争

# 除了摆在明面上的斗争之外，暗中的各种竞争也无处不在。
# 竞争的存在与阶级类似，同样是可以被量化的数据在人类群体这个范围内的产物。在着手解决某一件事情时，我们总会希望在总体上拥有更好评价的个人来负责，这种行为增大了最终结果向更符合群体需求的方向发展的概率。如果要增加人手，我们也更倾向于在剩余的候选人中再次进行最优的选择。这种客观的需求在不掺杂个人特定的情感因素之时效果良好，也是整体自然而然的选择方向。
# 在数据可以被进行量化以及在某个范畴之内的个人有限之时，这就成为了竞争形成的充分条件。
# 万类霜天竞自由，在个人的生物性方面我们也早已与其他生物在进行生存环境的竞争。这种竞争在早期的激烈程度也称得上是斗争，因为早期在生存方面的需求所有的生物都不能进行让步，而能满足生存需求的资源也没有达到现在的丰富度。为了个人生存的各种流血斗争，在任何时间对于个人来说都是合理的需求。
# 在个人的生存需求得到了满足之后，这种竞争的激烈程度也会得到下降，成为了一种温和斗争。这种温和斗争在性质上我们可以进行一次划分：良性的竞争与恶性的竞争。在良性的竞争环境中，所有参与这种竞争的个人首先在这种竞争中不会丧失自身的生存权利，这让竞争者在竞争之时没有了失败的后顾之忧。同时这种竞争也应当是多个方面的共同考察，而不是单一方面的数值比较。这种良性竞争在时间的作用下使得所有竞争者在不同的方面都各有所长，在不同的层面的数值需要中都能找到更加符合的候选人。而恶性的竞争则总体上会对整体产生消极作用，在了解之时我们也需要尽量避免。这种恶性竞争既有可能是参与者的情感之中包含恶意，也有可能是这种竞争会造成资源的浪费。在自由市场中，特定的主体在利用各种手段之时可以让公共概念空间中竞争对手的整体评价在短时间内下滑，从而让自身的竞争力在相对性中得到增强。在以筛选为目的导向的各种考核中，对于有限空间的竞争让所有候选者在达到了比较优秀的表现之后仍然需要付出较大的代价来换取少量微不足道的提升。对于整体来说，如果个人将这些努力放在其他方面的话在很大的概率下能产生更多的效益；但对于个人来说，在这种明确的目标之下大部分参与者的选择都是继续内卷。
# 既然我们都知道良性竞争在整体上来说是有益的，而恶性竞争在整体上来说是有害的，那为什么这里仍存在恶性竞争？
# 不同参与者互相针对的恶性竞争来源于其成本低廉而收效明显。在不需要个人提升自身竞争力之时对竞争对手进行不符合实际的宣传或者是选择性的说明，都在削弱竞争对手的竞争力之时变相地提升了自身的竞争力。对于这种竞争我们需要提升这种行为的成本，增加这种行为的惩罚。竞争参与者在综合成本与其获得的利益之时如果为负数，那么也就失去了这方面的兴趣。另一种筛选过程中的参与者，在所有竞争者都不愿从中滑落之时，个人陷入了这种无效竞争的漩涡，这种外在表现也是多方面因素共同作用的结果。在水流系统中，水会向压力最低的地方流动，而由个人组成的整体也具有这个方面的趋势。在具体的方向上已经产生了明显的恶性竞争，那其他方向的情况恐怕也不容乐观。
# 而我们的目的是什么？竞争本身并没有自身的问题，在各种条件下所造成的错误应用才是应当解决的问题。如果我们能给予所有参与者足够的自身空间，优化整体的评价体系并在这个基础上发挥每个个人在良性竞争之后的所有潜能，那么这也就是相对理想的一种设想。在各种利益的影响之下，每个参与者在大胆假设与自身的抉择之下，各种在不同程度上偏离的选择成为了最终的选项。
# 个人在做出自身的选择时，也需要去了解自身与竞争之间的联系。对各个方面有了更多的了解之后，整体中的个人的提升也让整体的竞争有了向更好的方向发展的迹象。
# 竞争经营着系统演化的基本生意。
