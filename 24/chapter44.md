# chapter44 政治关系

# 外部交易的过程也会受到政治关系影响。
# 在权力概念的讨论中我们就已经涉及到政治关系，但这里会更加偏向于国际之间的政治往来。来源于政府内部权利斗争的政治手段，在外部也具有参考作用。而对于不同的主体，也应当采取不同的交往手段。
# 我们可以继续思考一下内部的政治斗争。在具体的解释中，政治的含义为对于管理与利益分配的实践过程，而权利的交替就是政治体系的瓦解与重新建立。如果在重新建立的过程中，所有人都成为了你的敌人，那么这种失败的政治关系也会导致无可挽回的结果。虽然个人的毁灭在实质上也只是物质的组成方式与外在结构发生了变化，但这却丢失了自然的幸运。
# 由政治关系而建立起的社会秩序是经济稳定发展的外部环境。在这个语境下，法律和道德作为秩序的强要求与底线约束，以政治关系的两种手段来形成影响。这种手段在解决不同的矛盾之时也是成熟的方法，红脸与白脸在软硬兼施的行为下也容易让参与者在同一事件上达成一致。
# 因为实行政治手段的主体与被政治手段所影响的客体属于同一层次的具体概念，在很多情况下都能产生政治地位的易位与重新组合，所以法律这种强制手段在这个方面也具有一定的自身意义。而道德作为个人情感的粘合剂，在个人脱离集体走向对立面的情况下也具有抑制作用。
# 回到国际之间的政治关系。外交作为主要手段，在国际上也被看做不同国家主体的对外政策的窗口。而不同的国家在某些问题上达成一致，既可能基于同一利益共同体的现状，也可能是地缘条件与认知错位导致的结果。具有相似产业结构的国家主体在两败俱伤这种竞争的可能结局的预测下，在双方的政治合作中达成了对于其他国家的一致。某些国家在树立了同一个假想敌之后也会传播这种观点，在这种并不准确的外在表现下形成的却是乌合之众。
# 在国际的政治关系中我们最熟悉的莫过于弱国无外交。在不完全竞争与有限范围的合作之下，使用模型进行模拟所能得到的结果大概率是少量的高竞争力国家与大量的低竞争力国家。这些少量的高竞争力国家之间是一种微妙的平衡，无论哪一方的竞争力的提升都会促进其他各方的合作进程。而在高竞争力国家与低竞争力国家的交互中，高竞争力国家所能拥有的优势显而易见。
# 首先可以来看一下低竞争力国家在客观条件下的自身选择。在物质世界的相对位置中如果其靠进某个具体的高竞争力国家那在正常的经济交流活动中自然而然会受到这种特定高竞争力国家的重要影响，在相对公平的政治关系的建立中也会更加倾向于这种高竞争力国家。这种空间上的距离相近也可能会导致走向相反的道路，但这种情况也遵循具体问题具体分析的逻辑，在相反的道路上对于基本的客观条件也会有相同的认知。而对于那些远离高竞争力国家的独立主体，在形式上保持中立则是一种安生立命之道。但这里却并不存在完全的中立，在现状的不断改变中所隐藏的方向选择也说明了墙头草的本质。
# 所以低竞争力的国家在特定条件下进行区域合作应该怎么看待？
# 如果这种合作的参与者不包括高竞争力国家，那么这种合作的动力也大概率来自于高竞争力国家的压力。判断这种动力的来源也很简单，在外部高竞争力国家不被纳入考虑范围之时，在合作的参与者中也就有了相对的高竞争力国家，重新选择一个高竞争力国家来占据先前的生态位的过程也就是个人进行有效决策的依赖。
# 这里也不得不提及一下以意识形态的先进性所组成的国家联盟。有着同一个目标，在同一意志所形成的本质信任下这些国家发挥出了主观能动性的重要价值。在这种政治关系中，利益分配与权利斗争这些让社会螺旋上升的因素失去了大部分意义，管理者在自觉性中的全局行动得到了完全的支持。
# 然而事情往往会出现意外，在众多复杂因素的影响下也证明了没有永恒的体系。但这种尝试在实际上已经证明了自身的可行性，无论是内部因素还是外部原因都不能掩盖这种行为的深远影响。
# 政治关系的博弈依靠更高层次的思考。
