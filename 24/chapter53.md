# chapter53 色彩

# 在对自身的研究过程中，三种类别的感光细胞在不同的刺激强度的影响下所形成的整体感受，就是色彩的不同组合。
# 以被三棱柱所散射的白光说起。在不同波长的电磁波被过滤出来进行单独分析时，特定频率的电磁波在不同的个人层面上有着类似的感受，将其对应到某个语义上时对这种事物的复制与再现过程也有所帮助。一一检视可以产生不同刺激感受的不同的频率的电磁波，在色彩的集合中我们也逐渐增加对应的概念。
# 在区分不同色彩的细微差别时，具体的自然语言不太容易进行区分。这既来源于自然语言偏向质的区别的特点，同时也没有必要。相关领域的个人在进行了一致性讨论之后，在公共概念空间中设计了不同色彩的编码标准。这种标准在偏向数量的特点上更加准确，也得到了广泛使用。在不同的设备都遵守同一种色彩标准时，交流活动中的色彩元素也就没有了阻碍。
# 以数量形式所表示的色彩概念，也具有自身的加法与减法特性。将不同的色彩进行叠加，我们可以得到另一种色彩，这也意味着得到的色彩可以分解成这两种色彩的和。将色彩空间沿着不同的方向进行切割时，这里的横截面所展示的变化图像会显现出相应的变化规律。在图像消失的边界处，则是不可见光的生效范围。借助工具来探测不同波长的电磁波之时，不可见光也能被捕捉到踪迹。这些波长的光我们接触得也并不多，所以我们的讨论范围也仅限于原生工具所能感知的区域。
# 现在就可以审视一下原生工具的生成结果。假设个人及其周围的上下文的状态不再改变，那么个人从外界所接收到的也就是在特定的观察方向是特定数量级的色彩集合。而因为原生工具的精度在足够满足自身需求之时就失去了继续深入的动力，这些色彩组成的集合在经过有限的完全复制后在另一个人身上播放也就能产生同样的图像。将时序性加入我们的思考进程，在持续的同样的色彩刺激中原生工具在动态调节下会减弱这样的刺激。这种现象在已经进入了减弱刺激的状态之时切换到另一种色彩刺激的时候可以比较明显地感受到，这种切换感受也根据进入状态的时间长度而不同。
# 与很多概念一样，色彩作为公共概念在整体的使用中也逐渐具有了一定的内在含义，不同的色彩形成的对比也加深了这种印象。当色彩偏向于蓝色这种冷色调时，我们会比较放松，感到松弛；而当色彩偏向于红色这种暖色调时，我们会比较兴奋，感到紧张。不同的色彩在一定程度上占据了不同的内在含义，在使用色彩组合时需要注意这点。
# 色彩之间的协调也是分析的子对象。将不同的色彩进行组合显示时，整体上的色彩的空间结构与这种色彩所具有的一定的内在含义共同决定了设计者所展现的实际效果，尽管这种效果可能与理想有一定的差距。在比较保守的思维影响下，参考客观事物的色彩选择方向并进行微调，在复制与再现的变种中达到不犯错的目的。
# 色彩的空间性的形成过程也很容易理解。在使用照相机模拟原生工具的生效方式时，忽略观察点的空间形状不会产生太多的偏差。而原生工具在物质空间中就占据了一定的空间，位于同一个相对位置的色彩在不同的接受器都收集到相关的信息时在大脑的计算中也就拥有了空间中深度的感知。
# 再加上状态变化过程的产生原因，我们就得到了流动的色彩。在所有的时间内，相对的位置已经固定了下来，发生改变的也只是相对位置上的色彩数据。也就是在这种数据的分析过程中，个人采用了一种高效的处理方法来关注重点，并构建其逻辑结构来进行处理。这个过程的步骤也并不多，个人大脑的运转效率也有限，在渐进式的迭代过程中也形成了一种设计的哲学。
# 使用色彩来进行思维交流，信息传递，我们对不同的事物产生了相应的理解。原生工具的多点接收，结合色彩数据形成了空间感知。状态的改变流程，则丰富了色彩的组合层次。
# 色彩在波长范围内形成了封闭的集合，可见光之内即是命运。
