# chapter19 梦境

# 在现实中增添一些想象空间，就产生了梦境。
# 在大脑处于清醒状态时，我们就在进行各种各样的认知实践活动。在个人的生理条件下，大脑不能一直保持清醒状态，我们需要睡眠来进行恢复。而梦境则是在睡眠中进行的一种意识活动，这种活动也往往与个人所经历的一些重要事件或者对某些事物的强烈感情存在密切联系。
# 对梦境的研究也一直在进行。无论是周公解梦，还是弗洛伊德的梦的解析，在这种模糊的梦境之上做出的解读，似乎没有任何道理。但是梦境也是人的潜意识活跃的表现，在清醒状态个人会在无意识中压制这些潜意识。
# 可以先从梦境本身看起，这里我先分析一下我个人过去的梦境经历。这些经历大多都是模拟在大脑清醒过程中所经历的场景，然后以第三人称的视角来查看自身在这个场景中的各种活动。而在梦境中场景的状态走向，往往和实际的走向存在差别。这种状态的走向往往会遵循潜意识的走向，也就是满足了个人的某种欲望。最简单的情况就是如果你在大脑清醒的阶段一直在去思考某件事情，或者你在这之上有着强烈感情，那在进入睡眠之后的梦境中你很可能用个人的潜意识实现欲望的满足。
# 这种满足个人欲望的梦境也是大多数梦境产生的原因。如果只是生理上的疲倦，纯粹的睡眠方式的休息并不会产生梦境。而在大脑所有的每次长时间的休息状态中，存在梦境的睡眠也不占多数。
# 个人与生俱来的的恐惧也是一些梦境的来源，这种个人不愿去回顾的噩梦也是潜意识的另一方面的体现。在梦见恐怖的事情时，这种恐怖概念的外在形态可能可以追溯到原始人在夜晚的火堆旁对黑暗的印象。
# 在这些梦境的表象上，我们可以来进行一些分析。这些分析可能会指向一个结果，而如果以结果为分析前提我们可以设计各种各样合理的论证过程，所以我们不必太过在意结果和论证过程。如果你在梦境中想上厕所，那么你如何去分析这个梦境？联系现实时在进入睡眠之前喝了比较多的水是一种可能的原因。而如果你在梦境中从楼梯上摔了下去，那可能是因为现实中你从床上掉了下去。
# 对于梦境也有一种象征意义的分析。这种分析使用替换方法，把梦境中的概念使用另一种隐藏的概念来进行替代，从而进行合理性分析。如果你在梦中遇到水，许多的象征分析会认为这是财富的象征，同样的，如果梦见了火则之后可能会遇到好运。这并没有什么科学的依据，但个人会有心理暗示。
# 梦境更像是一种个人的虚拟世界，在大脑的活动中你可以存在各种各样的想象，然后在梦境的模糊视觉中引入现实。在这种梦境中你已经知道自己身处梦境，而我也没有在已经知道自己身处梦境中时进入更深一层梦境的体验。
# LSD梦境模拟器作为工程师将梦境引入现实的一次尝试，描述了梦境的不连续性和模糊性。对于某个特定的梦境，你不知道哪里是这一场梦境的开端，而在后续的走向中你也不知道会在什么时候会被打断。在梦中出现的事物也都是模糊的，色彩和形状属性是主要特点。而在这个游戏的最后，各种迷幻的图片留给人们的是无尽的想象。而LSD,则是一种强烈的半人工致幻剂。它无色无味，就像清澈的纯水。在LSD所制造的幻象中，它打破了个人认知的框架，呈现出更加鲜艳多彩又天旋地转的世界。
# 当我们从梦境中回到现实时，在试着回忆起梦境中所发生的事情时，我们也会发现很快就会遗忘。而之后我们也只会记得今天做了一个梦，然后在梦境中好像发生了什么而已，这种遗忘也是个人忽略不重要的部分的表现。
# 当我们回到了一层现实时，你怎么知道现实之外还有没有另外的现实？在不知道做出当前现实的梦境的主体时，我们所见的清晰影像和思考活动也只是主体视角下的一种梦境。当主体离开梦境之时，我们也会随之消失。但现在我们不会知道主体会在什么时候回到现实，这是一种幸运还是一种不幸？
# 梦境也是另一种现实。
