# chapter56 组合

# 事物在具有自身内部结构来作为整体参与外界行为时，本身也可以进行更小范围的划分。
# 组合关系在物质层面与思维层面都发挥着自身的作用，而所有可知的事物都经由组合关系与其他概念之间进行联系。首先可以对于物质层面的组合关系来讨论一下。建筑在不同的文化中有着不同的外观呈现，也是组合关系的典型体现。搭建具体的建筑之前，我们需要先进行相关材料的准备，这里可能包括木材，沙石以及各种粘合剂。从平地上开始的搭建工作，需要先打好相应的地基。地基作为整个建筑对于外界的依赖，需要审慎的检查工作确保无误并在后续中不会发生修改。在地基打好之后，按照设计图纸进行的施工也就可以开始。在一层层向上填充的过程中，各种建筑材料也被用来在空间中形成相对联系的基本元素。这些基本元素之间的联系有的处于关键位置，一旦缺失就会对整体的功能造成影响，而部分联系的缺失在短时间内也并不会导致整体功能的中断。一个个版本迭代下来的设计蓝图，在相关信息的逐渐丢失中也丧失了自身的修复能力，后续的沿袭过程在最小化犯错的可能性时也让修改变得更难，阻力也就更大。增量修改在不改变原先结构的情况下完成了个人的需求，也只一种普遍做法。
# 组成个人的所有物质也有着相同的规律。不同的是，组成建筑的元素在建筑落成之后基本不会进行主体的替换行为，而在多次的物质交换中组成个人的元素也在进行着动态变化。这样的动态变化导致了在不同时间点进行个人的元素分析时，管理者会发现绝大多数的编号都不太相同。不会发生变化的部分保存了个人的原始信息而成为了个人的唯一标识，但也只是有着相对更长的作用时间而已。
# 庞加莱回归定理作为数学范畴中的一个定理，确定了在有限的测度空间内，存在着无穷多个可回归点，这些可回归点的阶次也从一扩展到无穷。数学上的定理往往更加理想化，但现实世界在满足定理的应用条件之时就在这二者之间建立了联系。对于现实世界我们的观测并不完备，在未发现反例之时可以先假设这个定理可以应用于物质世界。那么这意味着什么？
# 在某个时间点处在某个相对位置的元素，在足够长的时间后会回到与原点之间的距离足够小的位置。随着初始位置与后续位置之间的距离缩小，回归时间也越来越漫长。在个人的短暂生命中，这种漫长的时间几乎没有任何意义，但好在是有限的。现在使用着组成自身的元素进行思考的我，随着元素的分开也就丢失了之间的联系所构成的整体上的思考能力。在下一次庞加莱回归之时，另一个相似的我会产生怎样的思考？如果更加注重现在，我们自身处在这种组合状态的哪一个优劣层次？
# 这种思考可能会让我们想到再现活动。但这里也并不会存在事实上的再现过程，庞加莱回归在更大可能性中也只是在遥远的某个时间点的一瞬间一个确定的范围与当下的某个时间点的相同范围无限接近，但之后就是全新的走向了。但这种瞬间的过程也足以让我们产生遐想，由内而外的一种强烈的感情也足以让人感动。
# 以事物之间的组合关系作为研究对象，思维层面主要将这些联系复制到概念空间中。概念上与现实的对应让这种工作也非常简单，也不需要进行思考活动。在这种对应过程完成之后，使用这些要素我们就能搭建出自身的思考过程。
# 再看一下我们所使用的所有思考组合，这里的大部分都来源于他人已经应用过的固定流程。为了制作个人层面上具有完整意义的具体设计，使用不同的积木进行全新的联系过程在大部分情况下都已足够。如果元素因为不具有完全的独特性而不能满足需求，那么个人也可以在下一个层次进行个人思维的投射过程，来搭建上层中具有唯一性的元素，最终应用于这里的具体设计。
# 相比于沿袭旧有的设计然后进行修改，将不同的功能进行最小化然后使用组合方法是更好的选择。但这也需要进行合理使用，如何进行完整性的认识与联系之间的同一性而进行拆分是需要个人审慎思考的流程。
# 组合在不同的层次上形成了复杂又完整的功能。
