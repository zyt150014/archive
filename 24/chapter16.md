# chapter16 语义变化

# 在不同的状态时，相同的语义会有不同的解释。而在同一个状态时，不同的语义也可能指向同一个概念。
# 我们可以先来看一下语义膨胀，这是人们在对于相对程度的不同感知。在公共的概念空间中，存在一批相关联的概念用来指代不同的程度。这里我可以列举一下：很糟糕，有点坏，一般，还可以，非常好。在通常的形容中，我们也都知道这些语义后面的程度大小。综合全体对某个概念的不同的程度评价，我们可以获得一个概率分布图。分布图的横坐标表示程度从低到高，纵坐标表示统计数量的多少。与统计学中常见的正态分布相比，语义的概率分布图一般会更加向更好的方向倾斜。在评价一种常见的食物时，出于个人的原因我们更可能去偏向于给出相对于实际更好的评价。而在这种情况下，一般也可能会更多地表示有点坏，还可以就取代了一般的含义。在一定时间之后，我们也可以重新建立同一种概念的概率分布图。在将新的概率分布图和之前的概率分布图放在同一个坐标系下时，很容易就能看出新的曲线偏向于更好的方向。但是客观上我们仍然需要不同的词语来表示程度的相对低和相对高，所以新的概率分布图也会重新按照程度的从低到高一一对应，而这个过程也似乎没有尽头。
# 这种语义的膨胀也对我们产生了一些影响。在人们使用了太多次good之后，新的good的语义逐渐变成了一般的含义，excellent开始变得常用。而在excellent变得更加常用之后，我们就需要新的语义来表示excellent原先表示的程度。这种情况可能会导致我们无法找到一个更好程度的概念，这也会在一定程度上阻碍个体之间的交流活动。
# 语义膨胀现象从本质来说，是人类心理学方面的原因。在客观角度下个人也没有什么解决办法，个人都是在大的程度下被整体影响而在小的程度下影响整体。
# 在这个状态下，创造一些新的词语用来占据新的程度语义比较常见。我们可以在所有的词语组合中寻找之前尚未被使用的组合，或者也可以使用一些新的字，不过后者的发生概率在很大程度上低于前者，因为现有的字的重新组合往往更加简单。而现实情况也是如此，新词的收录次数也比不上各种新的习惯用语的产生，而新词在某种意义上也是可解释的，也只是一种习惯用语的缩略形式而已。
# 除了各种新概念的产生导致的新的词语组合的形成之外，也有另一种替代语义也会推动这个过程。屏蔽词也是在日常生活中比较常见的语义变化现象，不过这是人为构造的规则而不是随着物质的状态改变而产生的指代新概念的语义，在这里我也仅仅讨论这种屏蔽词而引发的现象。
# 最容易被屏蔽的大多都是一些情绪宣泄的概念语义，各种人身攻击和冷嘲热讽等。而在因为某些原因将这些词语屏蔽之后，人们客观上的需求却并没有消失。自然而然使用其他类似或者能引发个人概念空间与先前语义进行关联的新的概念的现象也开始出现。这种替代作用也可能会污染原先概念的语义，然后会在某个时间引发意想不到的联想。另外的屏蔽词也就是更加人为的原因，导致这些词语直接被封禁。如果想要在字面上表示这些屏蔽词的含义，也需要前面提到的个人空间概念中的关联的作用效果。屏蔽词的增多也并不会改善这种现象，个人会去采用各种在不明白的人看来莫名奇妙的词语组合来表示被屏蔽的语义。
# 而在具有各种不同语义的语义集合的群体之间进行交流活动时，直接引入对方的语义以及语义之间的关联是符合客观规律的，这也是各种外来词的形成过程。外来词会增加新的语义，然后在状态的变化中逐渐融入原先的语义集合。这种借鉴可以在语义中保留引入过程，同时也不会增加理解成本。
# 随着不同的状态的改变，语义也会随之变得不同。在过去的语境下的词语所承载的语义，也需要在指向相同语义的词语发生变化后重新对应起来，这也就是历史文件不具有现实意义的一种解释。
# 语义变化影响着概念空间的对应关系，我们也需要在变化的世界中进行适当改变。
