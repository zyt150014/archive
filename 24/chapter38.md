# chapter38 外在表现

# 同样的是，在艺术的创作中，外在表现是重要的参考条件。
# 外在表现也很容易理解，这就是个人感官所能感受到的所有信息。无论是视觉上，听觉上还是其余的知觉，这些信息共同形成了一个多角度的物质世界。
# 我们可以先谈论一下视觉上的外在表现。依据当下个人对于世界的观察过程，眼睛的构造是用来接受不同颜色的光从而产生一种混合刺激，最终在大脑中产生内在的印象。而光的颜色属性本质上是光的波长或者说是频率属性，感光细胞对于不同频率的光产生不同的反应，并将这些信息以神经传导中的电信号的形式进行传递。
# 分析不同频率的光之时，在个人的角度可以将其分为可见光和不可见光，产生感知从而形成外在表现的主要是可见光，而使用特定工具我们也可以观察到不可见光所造成的不同层面的影响。
# 牛顿使用三棱镜将白光分解成了不同颜色的光，这扩大了我们对于外在表现的知识边界，也体现了不同的外在表现的叠加原理。这种逻辑与内在含义也是相似的，使用同一类概念的不同方面的表现，并在不同程度上进行结合，然后在不同的占比与相对结构中形成艺术的外在表现。
# 而这种外在表现的记录也可以去思考一下。在影像记录方式得到发展的现在，我们随时都可以将在某个时间的某个空间位置将某个方向的视觉感受数据在不损失大部分信息的同时能够得到长时间的保存，并在后续的某个时间点复现出类似的视觉感受。在此之前，借助文字或者符号等信息量较少的传递载体在我们对外在表现进行恢复时则具有很大的裁量权利。当我们使用文字时，个人的想象能力对于未描述方面的补充虽然可能和实际的状况存在一定的区别，但这也是一种丰富外在表现的有效方法。
# 使用记录载体记录外在表现之后，随之而来的就是使用这些记录数据来按照某些规则进行二次创作。将原始的客观物质世界的外在表现当作第一层，那么记录载体则在编码过程中会天然地损失一部分数据，从而获得第二层的结果。在后续的过程中无论是进行色调调整还是风格迁移，都可以看作是第三层的结果。第二层和第三层在本质上同样都是数据编码从而被用作特定工具的解码原料，其不同之处在于第二层直接来源于近似模拟信号的物质世界的运作，第三层与第二层在同属类似数字信号的编码中随着处理方式的变化不一而足。
# 这也让我想起了个人先前的设想。这种记录的编码在一定程度上只是所有数据组合的一种方式而已，我们是否能在所有的组合方式中找到对于人类有认知意义并且具有内在含义的组合方式？
# 这显然是具有一定可行性的。在这种艺术的创作过程中，个人首先确定具体的内在含义，并理解表达处这些内在含义所需要的外在表现，这些外在表现也就可以参考事物自身内在含义所具有的独特意义用来迁移同种事物的外在表现。在人工操作缓慢的情况下使用计算机来快速迭代，所输出的结果则是这个状态改变过程中的艺术成果。
# 但这种外在表现在不成熟的时候我们很容易看出瑕疵与痕迹，这种模板式的风格可能也只是发展过程中的实验品，并不足以达到以假乱真的地步。但是这也是一种担忧：如果外在表现可以被完全模拟，视觉上可以完全替代物质世界的感受，那就走向了缸中之脑的第一步。
# 外在表现也存在一些其他的问题，也就是外在表现与内在含义也并不是完全对应的关系。在个人看来外在表现仍然没有区别的两种情况，其内在含义则在状态的改变过程中也在进行变动。就像正在工作的电子机器一样，在个人不进行操作之时这种外在的显示在一段时间之内也并不会变化，但内在的电流，温度分布以及信号传输等数据也不会完全相同。将同一个内在含义进行不同的外在表现包装则更加被我们所熟知，这也就是各种换皮事物所采用的原理。
# 在听觉这些小部分的知觉看来，这方面的外在表现的研究则不如视觉感知详细。而外在表现的发展方向也是沿着全感知的方向，显然这里还有很长的路。
# 外在表现又回到了对于真与假的讨论，我们需要正确看待。
