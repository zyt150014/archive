# chapter45 改革

# 在不断演进的政治关系与经济生活中，这二者之间的不适配会引出改革的需要。
# 以政治的角度来看，这也被成为经济基础与上层建筑。这二者在改革的作用下首先关注二者的接触方式，然后在这些接触点处进行或多或少的改动，从而让这二者之间的传导路径保持畅通。在深入这些改动时，这里也有几种不同的修改方式。其一：进行较多的政治关系的修改与较少的经济生活的修改；其二：进行较多的经济生活的修改与较少政治关系的修改；其三：都进行较少的修改并引入中间人的概念连接二者。
# 在第一种方式中，这承认了经济的重要地位，在保持经济框架的前提下因地制宜，这种延续性也会让经济在稳定中持续发展。经济要素作为个人实现的基本条件，主体在不需要过多的干预之下进行适当引导即可发挥出大部分人的能动性，从而满足经济发展的充足需要。
# 在第二种方式中，这种决策者对于政治关系的自信让最终的选择落在了这个范围内。对于客观的物质世界的不够了解，让这种方式在解决实际问题中并没有发挥出什么作用。就像头痛医脚一样，不能找到进行实践活动的方向那么也显然会无所作为。这种情况在状态改变的累积效应下不断恶化，所需要的手段也可能从改革到革命进而缩小到最后的推倒重建。
# 第三种方式则偏向于平庸与温和。首先，这种不适配在大概率的情况下这二者都负有一定的责任，在多个具有独立思考的主体的所有行为所组成的整体上也更加如此。基于这种考量而施行的这种方式是一种保守的做法，当然这在尽量少犯错的层面上是正确而偷懒的操作。
# 这些关注二者交集所提出的解决方案在实际执行时会有自身的一定效果，但在不修改本质的情况下也就是治标不治本。考虑这个范畴内与政治关系和经济生活内部的综合形势，提出的一种包含各方面思考的可行的操作流程，并能让参与其中的执行者在互相配合中有条不紊地运作，在这一点上对决策者提出了很高的要求。
# 所以这种深度改革很难去推动以及在最终有一个比较高的完成度。决策者在充分的认知与详尽的思考之后，拿出了一份比较令人满意的答卷。执行者在流水线的操作中会遇到各种各样的阻碍，在决策者排除客观条件之后这些阻碍也就大多来自于利益不均的个人。兜兜转转，我们又回到了对于利益的讨论。
# 在利益上的博弈在改革这个话题下也没有太多的讨论价值，个人在坚持自身的情况下也失去了对于整体的思考，而如果这种选择来自于先前的事件所积累的不信任情感那么也就情有可原。这种情况我们也有相应的对策，也就是利用周围环境的逐步改革带动整体的改革进程。
# 不断变化的物质世界，政治关系与经济生活所受到的影响，也催生了对于改革的持续需求。在这个前行的过程中，改革作为掌舵人的桨，在个人的判断所形成的罗盘指引中足以不断调整自身的方向，在大海上逐渐接近自身的目标。
# 与引起剧烈社会变革的革命相比，改革这种渐进式修补策略既能达到维持经济生活与政治关系的平稳演进，又能落在大多数人的接受范围内。如果改革不能显现出自身的成效，那积重难返的现状也需要进行更大范围的行动。这种斗争也往往需要更大的牺牲，这种斗争换来的结果也值得革命者的这种付出。革命造成的深远影响也符合新旧事物的替换规律，因循守旧而产生的对于新事物的或轻视或恐惧的情感体验也只是历史长河中消失的一片浪花。
# 但这种革命需要进行谨慎的使用，错误的使用方式会反受其害。上个世纪和平年代所推行的革命性质的全体活动，在特定的历史条件下产生了沉重的后果。尽管推动这种革命的初衷是好的，但这种好的方面所造成的坏的结局则是我们不愿看到的。历史进程也已不可改变，如果能在执行时采取更好的方式也许是一种更优的选择。
# 改革带来了所有的希望。
