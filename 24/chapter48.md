# chapter48 替代

# 在个人与个人之间存在竞争关系时，被替代也是一种可能。
# 替代发生于拥有选择权的主体在下一级的多个选项中的选择行为，这是从无到有的过程。如果在主体已经拥有对应生态位的选择结果之后，在某种需求之下找到了一个更具有竞争力的选项，那么这也就是从有到优的过程。从无到有偏向于竞争方面的替代，而从有到优则注重于新旧之间的替代。这两种替代方式也很常见，同样也是一种客观规律。
# 这里我们也可以思考一下参与竞争与替代过程的主体。以是否具有主观性来划分的话，可以将主体分为能进行独立思考的生物与可以被生物所利用的客观事物，在个人的语境下简称为人与物。划分之后可以得到三种不同种类的关系：人与人之间的关系，人与物之间的关系，物与物之间的关系。竞争与替代也在这三种关系中不断流动。
# 在人与人的关系之中，在先前我们也已经提及到竞争，除此之外基于共同利益的合作关系也是重要的组成部分。在系统的复杂度不断上升的情况下，个人也逐渐无法去深入到每一个细节。在不同的个人利用不同的优势在互相的交流不存在障碍之时进行分工合作是更好的选择。如果个人对竞争没有兴趣，找到自身所认为的更有价值的事情也是另一条道路。
# 而在人与物之间的关系中，主要以个人利用主动性进行的实践过程为主。个人在改造客观事物的过程中逐渐将自身的主动性进行外溢，在各种工程中推行自动化流程，将个人所思考的结果使用客观事物进行记录，并利用后续过程中的重新展现来处理同样的事务。随着一层又一层的工具被记录下来用来解决特定的问题，将这些工具进行组合也是事物发展的最佳实践。
# 在提到物与物之间的关系之时，以一件事物对另一件事物的替代为主，而这种替代关系也来自于人的主观选择。在人的实践过程的不断发展之下，掺杂了个人主观性的物作为主体与其他完全客观的物之间的主动与被动关系也逐渐变得更加重要。这种关系在人与物关系的工具组合过程中也拥有很高的重构速度，相应的演化方向也需要进行重视。
# 我们在接受了人与人关系中存在的人替代人的过程之后，能否认识到在其余关系中所可能存在的物替代人的过程？
# 首先人与人之间就存在先天本质的不同，这无法避免。在不同的外部环境的影响下在某个时间点的切片中进行的量化分析可以在不同层面上对个人赋予不同的数值，从这里可以进行继续推导。具有更高数值的个人与相对较低数值的个人在进行同一类实践活动中所提出的个人方案也显然存在优劣之分，在这二者之间进行了更优的选择。具有更高竞争力的个人在与客观事物的交互行为中将自身的特性进行了传递，这也让外物在某种意义上也具有了自主性。与此类似的各种过程成为了更广泛的发展的起源。
# 在个人进行这种客观事物的自主性的累加过程中，也就是在使用各种现成工具进行越来越复杂的组装过程，这种量变过程也总会在某个时间点转变为质变过程。这种过程的推进在第一次工业革命之时也就初具雏形，低下的劳动效率在蒸汽机与纺织机下相形见绌，物也开始了自身替代人的过程。
# 个人在这种过程中也逐渐利用并熟悉新的事物，得以继续在此之上进行更高层次的组合活动。在迅速的发展中，所组合的事物越来越多的同时，每种组合事物的使用成本也在下降。这种对于自身的影响也是效率最大化的方向，这种演进也似乎没有尽头。
# 物替代人的过程以替代个人的重复劳动作为切入点，将个人从并没有更大意义的重复劳动中解放出来，从而进行更高层次的劳动活动。更高层次的劳动活动也会逐渐替代个人在各种不同方面的劳动，如此往复。在这个必然的过程摆在我们面前之时，个人也需要在自身的独立思考中不断改变自身的认知。如果主体在某个时间点被自身所创造的客体完全替代，在到达这种结果的过程中我们也只需要构建自洽的逻辑过程也就足够。
# 替代既是旧的结束，也是新的开始。
