# chapter51 复制

# 使用原料进行结构与含义上的模仿，是复制的一种普遍含义。
# 从具有独立思考能力的主体的新旧替换方式来看，这种复制活动也是实现这个目标的主要途径。研究对象在自身的生理规律下的有序度的缓慢下降过程中，找到了复制方法让自身在已经改变的状态不会完全重复的情况下获得了特定意义上的重新来过的机会。将记忆重置，以新的状态在先前所制造的积累中，个人在更大的概率性中能走得更远。生物学上的复制行为在具有独立思考能力的主体对自身开始进行了解之后会产生改变现状的思考活动，这种尝试也是一种全新的劳动方向。一种自然而然的思考也涌上心头：在生物的发展演化中，如果了解了复制方法的具体效果，有没有机会实现保留记忆的重新来过？
# 如果当下这种进化过程仍然没有达到终点，那么这种思考可能就是后续的方向。以现在的视角看来，相比于保留记忆，重新在白纸上进行实践活动也具有更多的施展空间。在一个已经建造了一半的房屋之上进行修改工作，基础结构的镣铐会影响修改的各种可能方向，这也就导致后续维护者的最优选择是顺应前人的路线，尽管这可能并不是整体所能达到的最好效果。
# 对于具体的概念，这种复制方式是否可行？这种基本概念就像建造房屋所需要的砖瓦，如果能预先对基本知识有一个了解，那在后续中只需要进行相对结构的搭建的工作就并不需要很长的时间。这似乎是一种可行的策略，为白纸提供了现成的墨水之时只待一副画作的诞生。但这也犯了一个逻辑错误，对于所有原料的认知往往需要结合具体的环境来谈，在这一步就已经开始了个人的概念空间的构建过程。
# 沿袭自身的复制方法，我们也学会了各种形式的保留过程。无论是最简单的文字复制，还是对于某个成熟项目的文档描述，都是个人自身的思考结构的一种体现。使用互相可以理解的媒介，复制过程也是主体之间互相联系的一环。
# 使用复制方法所制造出的备份，在不同的个人需要中进行分发可以解决原本在某个时间点进入竞争状态的问题。这种用法也非常普遍，作为复制体的同一种需求也经由复制来实现。谁拥有了原本，谁也就有了复制权利。版权在公共概念空间中作为全局变量也有着自身的唯一性，对于我们这里也并不存在另外的一个公共概念空间。
# 而这种版权首先就存在争议。原本的制作过程来源于状态的改变，进行声称的个人也很难确定具体的前后时间点。而思考是自由的，所有人都能使用自身的思维工具产生相应的结构结果。在同一个客观事物的研究中，不同的人使用不同的描述方式对同一种规律的阐述，在整个发展过程中也都具有自身的贡献。牛顿和莱布尼茨在各自的思考中以不同的形式阐述了数学上连续变化中的内在逻辑，并在现实世界中产生了深远的指导意义。后人在这个过程中也对这个规律进行了完善与修补，在众人的共同努力下这种成果也由所有人所共享，但我们也将公共概念空间中的命名权来使用到纪念这两位具有主要影响的数学家身上。
# 如果是个人的思考结果，或者是工作成果，在具有个人的独特性中也具有一定的复制权利，这里我们也需要涉及到许可证的概念。在能拥有版权的条目下，个人在从头进行开发的过程中也需要使用各种工具。而除了这些客观的工具之外，他人的思考结果也是其中一部分。在不同的许可证中，个人所拥有的权限也不同。在参考了他人的实现方式之时，我们首先需要遵守相应的许可证限制，同时也需要有自身的独特性，基本相同的结构并没有多大程度的启发意义。
# 这在工具上也是一样。对于同一个工具的共同维护，让所有贡献者和使用者都能从中受益。出于个人原因或者是政治因素在某个时间点制作的副本，在不加入后续的修改之时也就停留在了原地。没有流动性的水也是没有未来的，在后续的状态变化中也必然会被抛弃。如果是来自于小众的内部需要，那么在整体上的落后也显而易见。
# 除此之外，将复制品与原本进行结合使用，在部分数据由于某些意外丢失之后整个系统仍然能正常工作。对这些意外情况进行分类讨论，在主从结构的设计中得以应对所有考虑到的情况。
# 复制属于客观的结构映像，这也需要主动的个人维护。
