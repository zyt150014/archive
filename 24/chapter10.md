# chapter10 虚拟世界

# 个人的概念空间，也是一种虚拟世界。
# 和物理空间不同，在个人的意义上，虚拟世界完全依赖人的感知而存在。当下我们能想象出的虚拟世界，和物理空间中个人所在的宏观层次所能感受到的环境有着紧密的联系。而以最具有代表性的RPG类型的游戏为例，其中所能接触到的所有概念都是可理解的。最常见的情况是直接对现实的物理空间概念进行还原，比如建筑，植被，河流，也包括有人参与的生产生活关系的构建。这种直接复制在以将物理空间无时无刻不作为接触对象的我们看来在加上了虚拟标签后可以平滑地从物理空间切换到虚拟世界。而个人将输入经过一层键盘和鼠标的转换，将输出经过眼睛接受的显示器信息转换，虚拟世界和现实空间就提供了几乎一样的输入输出。
# 最简单的复制物理空间而成的虚拟世界是不吸引人的。如果虚拟世界和物理空间完全一致，那么个人就没有在输入输出流水线中增加管道的需要，直接接触物理空间所经过的路径更短。这种实现也基本上很少使用，在此基础上增加各种物理空间没有的概念，完善虚拟世界观，现在看来是所有人的物理子做出的选择。
# 然而我们为什么需要构建虚拟世界？
# 在现实的物理空间发展的过程中，毁灭和重生不断交替进行，这种过程的产生原因在个人所在的宏观层次有着各种各样的解释方式。物质是有限的，虚拟世界的范畴是无限的。在现实的物理空间中，我们必须遵守客观的规则，在各种限制约束的镣铐下跳舞。与此同时现实的物理空间和个人对美好世界的想象也不一样，物理空间在规律的演化下物质的走向和个人需要的方向也不同。作为主体，个人也在发挥着主观能动性将物理空间朝着组成个人的物理子的选择方向进行改造。
# 这种改造相对于人的生命长度来说进展缓慢，而对于虚拟的美好世界我们则可以尽情想象。冬日街头卖火柴的小女孩，在现实的物理空间中失去了对人世的希望，在划掉的一根根火柴的火焰光芒中进入了个人幻想的美好虚拟世界。在进入的那一刻小女孩也明白这种美好并不存在于物理空间中，但是她还是选择保留这一份现实和虚拟交错的想象片段。现实空间存在的以是否满足个人需要为标准进行分类的不美好的方面，在虚拟世界中可以对美好的方面进行完全实现。
# 极乐世界也是一种美好完全实现了的想象中的虚拟世界，从古至今都会让人产生向往。这种对于美好虚拟世界的想象一方面是个人改造世界的动力，另一方面也有陷入其中的可能。
# 对于飞行的想象，让我们在物理空间中也能使用飞机这种工具感受高处的视角；对于潜水的想象，让我们也能使用潜水艇这种工具进入深海。在将虚拟世界经过现实的物理空间的概念进行复制后而在此基础上增加的组装概念，我们在将其进行现实的物理空间的规律和约束化改造。
# 在盗梦空间的结尾不断旋转的陀螺，在将其作为判断虚拟世界和物理空间的标准之时，导演并没有给出一个准确的答案。如果是因为过于强烈的情感，我们也可能将美好的虚拟替代破碎的现实。而如果真的能从现实逃离，在美好的虚拟世界中度过终生，你又会如何选择？
# 让我们设想一下久远的未来。在各种方向上的发展越来越受到物理空间的限制之时，欺骗大脑的美好虚拟世界有着强大的吸引力。每个人都能将自己的大脑放进培养液中，各种计算机接口也与之相连，可以模拟出任何美好的虚拟世界，并且需要的能源相比于思考在现实物理空间的优化方案要少得多。在这个状态时，不再产出任何文化产品，我现在写下的文字也是尚未达到这个阶段的表现。人们无所谓悲伤和痛苦，只有无尽的快乐。我不去评价这种未来是好是坏，这也只是物理子的选择游戏的一种结果。
# 你也可以在现在回顾一下过去，看看自己所有的选择。今后你也可以使用虚拟世界拥有另外的身份，但你仍无法避免现实的物理世界的影响。
# 虚拟世界的美好，从现实的物理空间而来。
