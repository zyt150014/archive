# chapter6 效率

# 对于效率的追求，往往是越高越好。
# 如何计算效率的多少，我们可能都会想到最直接的公式：P = W/T。这里W可以是任何改造世界的行为，你可以选择看一本书，打扫一下房间，或者是出门游玩。W用来计算结束时间和起始时间这两个状态之间的所有差别，T是这两个变化之间所消耗的时间。但不同参考系有不同的时间度量方法，选择自身作为考察主体对于个人而言才有实际意义。在这个基础上，如果我们在同样的时间内，我们对世界的改造越多，那么我们的效率就越高。另一方面，在实现同样的改造结果之下，如果我们花费的时间越少，效率也同样越高。
# 对世界的改造可以简单地划分为有效改造和无效改造，也包括反向改造，这种划分逻辑也是依靠这些改造在全体人类的视角上的加权和。建造往往比破坏要难，这既是世界在走向无序的规律体现，也是我们在计算效率时需要考虑的问题。反向改造和无效改造让世界并没有变得更好，我们需要先把握住方向再去讨论如何行动。
# 在现实层面上，物品的流动效率也值得我们进行关注。道路的建立提升了物品到达所需之处的效率，也是社会运转的发动机。在这个多层流水线的结构下，整体效率的提升取决于效率最低的环节。而如果上游的效率较低，那么对下游的效率改造相对而言就比较低效，这也是一种判断效率高低的行为。个人的生产生活依靠物品的流动，而物品的流转速度也是经济发展状况的一种显示。
# 对于个人而言，现实总是存在各种各样需要去解决的问题。对于某个特定的问题，需要的时间是不同的。如果你在一个问题上花费了更少的时间，那么你就有更多的机会去进行思考，或者解决剩余的问题。在这里我们需要停下来看看，我们选择思考还是解决剩余的问题会产生哪些结果。你可以选择思考，来形成新的认知概念，构建新的实践方法，或者进行放松，这也是另一种放空思考；你也可以在提升效率之际完成更多的任务，迷失在工作的流水线中，逐渐成为庞大机器的一颗螺丝钉。这里也很容易看出我的感情倾向，我也只是觉得思考是人不被工作异化的原因。在实现被改造的客体可以模仿主体自身进行协作的过程中，我们需要更多思考产生的创造性工作。
# 但效率的提升也并没有这么简单。W首先也有许多影响因素，包括完成W的主体的各种客观条件，完成W所使用的各种工具，对W的各方面结果的评估，都无法忽视。与此同时这些因素之间也存在关联，也和“在你解决了一个问题后，因为这个解决问题的行为又产生了更多问题”相似。而如果要以现实观点对T取值，这里不能给出满意的答案，但我们往往会使用普遍认知来代替。一个简单例子：在个人完成一个任务之后，开始另外一个任务之前，先前的任务已经被这个人处理过的信息是开始这两个任务之时这个人的客观状态无法实现一致的根本原因，这也说明“人不能两次踏入同一条河流”。效率的提升也存在正反两面，在大脑进入状态之后也是注意力和精力不断消耗的过程。
# 如果我们再次从生命的开始到生命的结束这段时间来看，公式里面的T是在客观上确定的值。如果我们想要增加人生的厚度，那也应当去提高效率。
# 效率提高的另一个问题是往往会带来W质量的下降。很常见的现象就是在内容生产领域，在初稿确定之后如果不去进行加工润色，那么在提高了文章输出效率的同时也带来了文章质量的下降。同样的，我现在正在写的文字也会在之后完善细节，检查谬误，使其满足个人的合格标准。另一方面，对W的质量要求也是扩大受众的重要因素，在最终计算影响力和消耗的时间得出的效率来说也很有可能更高。
# 你可能也会问：通过提高效率而剩下的额外的时间，在思考之后可能并没有什么作用。而“人类一思考，上帝就发笑”也以一种轻松的方式面对这个问题。我也不会完全确定这对于个人来说是有用的，但是这至少让你有了多掷骰子的机会，前进道路上能走得更远，在预测的未来到来之时，能有更多的选项。
# 效率的方向，也是梯度最大的方向。
