# chapter23 自由

# 在谈论自由时，我们在谈论什么？
# 能选择做什么不是自由，能选择不做什么才是自由，不知道在什么时候了解到的语句也有它的局限性。在广义上，自由是有独立思考能力的个人在个人的对自己的行动和思考不被他人所限制的权利。在这个语义上，自由也是我们之前所说的一种力量，可以被法律所保障，同时也有被侵犯的可能。
# 人身自由，是个人在物质空间中自由的体现，也是个人进行认知和实践活动所必需的。随着状态的变化，我们的空间相对位置也在变化。如果个人被限制了人身自由，那么就意味着个人的空间相对位置在一个范围内。但如果将视野扩大，个人的生命是有限的，在世界的移动速度也是有限的，那个人的一生所经过的空间位置可以被某个空间范围覆盖，范围内即是命运。
# 但这也不是人身自由的含义。客观上，我们所有的任何自由都有范围，这基于我们的含义是有范围的。组成个人的所有物质在一定的时间范围内才被称为“你”，而这也是自由的范围的来源。如果在空间上你可以不选择到某个相对的地方，那这就是公共概念空间中的人身自由。
# 但人类社会有着复杂的社会秩序。除了上述的某些相对的地方，地球上也有许多人为的限制。军事禁区，政治地区等，和客观的恶劣的自然环境不同，这些地区的后果自负，个人的生命长度可能在此终止。对于恶劣的环境我们可以借助外物到达，无论是炎热的沙漠还是冰冷的南极地区，甚至是作为卫星的月球，都留下了人类的足迹。如果我也想到达月球，在客观上我是有这种自由的，但是作为普通人的我没有这种能力，这也是自由之外对个人还有一定的要求的体现。
# 在物质世界中的限制，我们可以在思想上不受限制。就像不同的视角一样，在第一人称视角的我在当前永远也不知道在第二人称视角的你在想些什么。你完全可以告诉我你在想什么，给出非常真诚的表情，或者是在科技继续发展之后能直接进行脑电波的交流，但在本质上，组成我的物质和组成你的物质不同，我们之间的交流必然存在交流渠道，而这个渠道所获得的信息的真与假也和先前所说一样无法去判断。如果存在一种判断方法，那这个判断方法也属于这个渠道获得的信息。
# 思想上的真正自由，只在于我们能不能思考到这个点。而随着状态的改变，人类所接触的概念组成的所有概念空间已经非常稠密了，这也就是在日常中你能遇到的所有概念基本上都有前人遇到过，只是你不知道这个前人是谁而已。而个人的大部分思想，也都从个人的认知与实践过程而来，这也就很难去有一种完全脱离现实存在的概念。如果你试着与他人交流这种概念，他人也不能理解你用特定语言描述的概念。
# 这也是一种幸运。这意味着我们可以随意地进行个人的想象，而他人也无法得知你现在在想些什么。而在各种想象之上，美好的虚拟世界则是借鉴现实概念而来的一种想象的现实投影。在这种想象中，虚拟世界的所有规则都由个人所设定，你可以使用各种无法解释的概念来组成你的所有想象。
# 这就又有了另一个问题。在作家的笔下，在导演的安排中，在游戏设计师的眼中，这些被设计出来的互相作用的概念并没有自由意志，完全依赖于主体的思考想象。在大部分的主体借鉴现实世界和他人的思考想象的作用之下，许多处于状态变化后续的想象和状态变化前的想象并不存在很大的区别。而为了在表面上进行区分，我们也会去进行一些包装。在不同的外表之下，是相似的内核。
# 这就需要我们有自由的独立思考，而这种思考才是组成你的物质之间的关联唯一的标志。我们每天都在与外界进行物质交换，每天醒来，今天组成你的物质的种类和数量和昨天显然不同，而昨天的“我”的概念就是随着昨天的思考和今天的思考的关联而和今天的“我”的概念产生了联系。利用思考上的自由，个人可以有一个完全唯一的人生轨迹。
# 在本质上自由的个人，也会有不同的追求与欲望。而在我们对这些的追求之中，我们也就很容易陷入不自由。这种思考的自由也是自由意志，主体才能以自己的意志来抵达结局。
# 自由的思考与想象之中，也是我们所有的创造。
