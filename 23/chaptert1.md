# Chapter1 认知

# 在永生和上传意识尚未实现的现在，人从出生就注定了生命终点的到来。在这段时间内，就是人认知世界和改造世界的过程。
# 为了更好地交流，语言被发明出来认知世界，但好像是一种比较低效也不甚准确的工具。让我们来思考一下，我们是如何解释语言的？我们可能都知道，在使用字典查找某个语言的意义时，是用同一种语言来解释的，这看起来陷入了自我解释的怪圈，但也是语言本质的一种体现，没有某一种概念能够脱离其他概念而单独存在。
# 让我们看下这句话：“天空是蓝色的”。 这句话有两个概念，一个是天空，一个是蓝色，还有一种认知方式。当我们提到天空时，如果使用“_是_的”的格式，或者是其他的格式，比如"_的颜色是_"，也可以是其他你愿意理解的方式，我们会将天空和蓝色这两个概念以某种形式建立起关联。而在人生的旅途中，我们首先利用现实世界在个人世界观的投影，在个人的概念空间中建立一层对应的象征表示。随着个人接触的世界越来越广阔，我们建立起的自身的概念空间也越来越复杂。在第一层的现实概念直接对应的个人概念空间元素之上，我们又逐渐构建起更加抽象的世界。抽象世界可以近乎无限地延伸，取决于个人大脑的能力强弱。完全与现实概念没有关联的个人概念空间的元素同样存在，我们大多将其作为某种模板或解决方法使用。这些抽象元素以有向图或无向图的形式存在，顶点表示抽象元素本身，边表示这些元素之间建立连接的方式。个人由此建立起的抽象世界大厦之间联系紧密，逻辑自洽，又随着现实环境的变化而不断发展。
# 上面描述了个体的世界认知过程。让我们扩展一下视野，看一下多人环境下的不同。在多人环境下，每个人都有与别人对比认知逻辑的机会。这里可以引入一个问题：在某种特定色盲m和正常无色盲的人n眼中，现实的特定颜色a，a具有特定的物理特性，被用于特定的行业，可以作为某种交通灯的颜色。在经过特定色盲m的个人世界投影之后，形成个人概念空间的颜色b，而在正常无色盲的人眼中会呈现出颜色c。这时在这两个人的交流中，两个人对于颜色b和颜色c都指向了一个公共的颜色概念d。这里先引入公共概念，后面再做解释。因为指向了同一个公共的颜色概念，所以这两个人在特定颜色a到公共颜色概念d这条路径上达成了一致。这会导致无法确定每个人的中间处理过程是否一致，这里会有某些意想不到的问题吗？在当下的语义环境下我们能明确确定吗？
# 除了个人的概念空间之外，还存在现实的公共概念空间。这两者之间的关系也很简单，就像“人的本质是社会关系的总和”一样，这种思考方式也适用与这两者。如果不存在个人的概念空间，那么公共概念空间便失去了意义。多个个人的概念空间互相影响，形成了作为事实标准的公共概念空间。
# 如何看待个人和公共的概念空间？我想以一个比较特殊的概念入手。对于“奥特曼”这个标记, 公共的概念空间会持有一份概率列表，表示这个标记在多个个人的概念空间中对应的概念所占的百分比。对于我来说，我可能会马上将其作为日本特摄英雄的总称，再思考一下我可能会联想到Sam Altman的中文姓，其他的概念的可能性就很小了。而对另一个人，如果在他的认知中有某个概念的外号是“奥特曼”，或者仅仅是更加熟悉Sam Altman的中文姓，那么我和他对于公共概念的解释的概率列表就会不同。这些概率列表的准确数值当然可以在某个时间点去精准量化，但这件事却并不是特别重要。个人对于某个标记的认知概率列表在不断变化，所以公共的概念空间的概率列表也无时无刻不在变化之中。
# 这种认知的不同也就很容易产生一些问题。当两个人用各自的概念空间进行交流时，误解和歧义无法完全避免。有点类似“夏虫不可语冰”，这也是语言这种工具不甚准确的原因。而我们如何去证明效率低下呢？我想正如之前所说，每个人的概率空间有不同的解释逻辑。我们使用声带发声，声波的传播速度在我个人认知的当下仅仅在百米每秒这个数量级，在达到另一个人的声波解释器的过程中还会不断衰减，声波的解释过程也容易出现误解，这会导致再一次的发声过程。显而易见，这种方式并不高效。
# 这种论证也不是完全正确，对于其中涉及的各种概念我们都可以有不同的认知。最简单的反驳，什么算高效？什么算低效？如果要准确地在数量级上去划分，我们无法去准确说明。
# 如果有更加高等的文明，我想个人的概念空间和整体的概念空间之间的歧义会更少，个人之间的交流更加频繁和快速，整体的概念空间更加稳定，多个人对于某个标记的概率列表更加趋同。再进一步，这里的误解和歧义能否完全避免？
# 认知作为其他所有人类发展出来的文明树枝的树根，这里也仅仅是作一些简单的思考，我们应该去了解更多细节。
